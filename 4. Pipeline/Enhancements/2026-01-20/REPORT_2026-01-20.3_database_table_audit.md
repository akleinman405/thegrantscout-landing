## Executive Summary

The TheGrantScout database (`f990_2025` schema) contains **38 tables** totaling **17.1 GB**. This audit documents each table's purpose, origin, and whether it should be kept, reviewed, or deleted.

### Quick Classification

| Status | Count | Size | Action |
|--------|-------|------|--------|
| ESSENTIAL | 14 | ~11.5 GB | Keep - Core pipeline |
| USEFUL | 10 | ~200 MB | Keep - Active use |
| DELETE | 4 | **~5 GB** | Delete - Redundant/archived |
| REVIEW | 10 | ~400 MB | Check if still needed |

**Potential disk savings: ~5 GB** by deleting confirmed redundant tables.
