## Investigation Findings

### REVIEW Table Analysis

1. **ref_org_name_lookup**: Contained 2.4M organization names from two sources:
   - `irs_bmf`: 1,921,700 records
   - `nonprofit_returns`: 508,155 records
   - Purpose was name matching for grant recipients without EINs
   - Not used in current pipeline - DELETE

2. **schedule_a**: IRS Schedule A data with public charity classification flags:
   - `church_ind`, `school_ind`, `hospital_ind`, etc.
   - No references in any `.py` or `.sql` files
   - Not used in current pipeline - DELETE

3. **calc_foundation_profiles vs calc_foundation_behavior**:
   - Both have 22 columns with ~80% overlap
   - `calc_foundation_profiles`: 143,184 rows (all foundations)
   - `calc_foundation_behavior`: 112,520 rows (only those with grant history)
   - Kept `profiles` because it has unique fields and more complete coverage

4. **fact_foundation_client_scores**:
   - Contains 12,652 pre-computed LASSO scores for 1 client (EIN 462730379)
   - Model version: v6.2
   - Last calculated: 2026-01-13
   - **Used in pipeline** at `phases/phase3_3_foundation_scores/05_aggregate_scores.sql` line 56-57
