## Phase 1: Threshold Qualifiers (Pass/Fail)

Before scoring, foundations must pass these gates:

| Qualifier | Criteria | If Fails |
|-----------|----------|----------|
| **Accessibility** | Application ≠ "Invitation Only" | → CULTIVATE track |
| **Minimum Grant Size** | Typical grant ≥ $25,000 | → EXCLUDE |
| **Basic Eligibility** | Status ≠ "NOT ELIGIBLE" | → EXCLUDE |
| **Decentralized Compatibility** | Can fund affiliate model | → EXCLUDE if explicitly cannot |

---

## Phase 2: Weighted Scoring (100 points)

| Factor | Weight | Scoring Criteria |
|--------|--------|------------------|
| **Track Record with Similar Orgs** | **35%** | Grants to youth-serving, education, federated nonprofits |
| **Geographic Alignment** | **25%** | 25 pts = exact Horizons city; 15 pts = regional; 10 pts = national ed funder |
| **Programmatic Fit** | **25%** | 25 pts = K-8 + summer; 18 pts = K-12/afterschool; 10 pts = general education |
| **Eligibility Strength** | **15%** | 15 pts = LIKELY; 10 pts = POSSIBLE; 5 pts = UNLIKELY |

## Phase 3: Score Interpretation

| Score Range | Recommendation | Action |
|-------------|----------------|--------|
| **75-100** | **SEND - High Priority** | Submit now; personalize heavily |
| **55-74** | **SEND - Standard** | Submit now; template with customization |
| **35-54** | **CONDITIONAL SEND** | Submit if capacity allows; batch process |
| **0-34** | **DON'T SEND** | Not a fit at this time |
| **Invitation-Only** | **CULTIVATE** | Relationship-building pipeline |

## Decision Flowchart

```
START → Is it Invitation Only?
        │
        ├─ YES → CULTIVATE (relationship track)
        │
        └─ NO → Is grant ≥ $25K?
                 │
                 ├─ NO → DON'T SEND (ROI not justified)
                 │
                 └─ YES → Is eligibility "NOT ELIGIBLE"?
                          │
                          ├─ YES → DON'T SEND
                          │
                          └─ NO → PROCEED TO SCORING
                                  │
                                  └─ Calculate weighted score
                                     │
                                     ├─ 75-100 → SEND High Priority
                                     ├─ 55-74  → SEND Standard
                                     ├─ 35-54  → CONDITIONAL SEND
                                     └─ 0-34   → DON'T SEND
```

## Invitation-Only Cultivation Strategy

For high-potential invitation-only funders:

1. **Identify connections** - Board overlaps, mutual grantees, program officer networks
2. **Invite site visits** - Program officers to local Horizons affiliates
3. **Monitor for shifts** - Watch for open RFPs or policy changes
4. **Re-evaluate annually** - May open applications

---

## Agent Consensus Points

1. **Accessibility gates everything** - Can't score what you can't access
2. **Track record with similar orgs is strongest signal** - Proves mission + size fit
3. **Geographic precision matters for hyperlocal strategy** - Exact city > regional > national
4. **Four-tier system beats binary** - Allows nuanced resource allocation
5. **Invitation-only deserves separate track** - Relationship building ≠ immediate application
