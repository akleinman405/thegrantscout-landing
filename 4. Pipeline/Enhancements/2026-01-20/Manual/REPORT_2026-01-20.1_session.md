## Hyperlocal Prospect Query - Data Sources

### Tables Used

| Table | Schema | Purpose |
|-------|--------|---------|
| `calc_foundation_features` | f990_2025 | Foundation characteristics, geographic focus |
| `fact_grants` | f990_2025 | Grant history for purpose keyword matching |

### Columns Used from `calc_foundation_features`

| Column | Purpose | Filter Applied |
|--------|---------|----------------|
| `ein` | Foundation identifier | - |
| `name` | Foundation name | - |
| `state` | Foundation HQ state | IN ('CT','CA','CO','DE','FL','GA','IL','IN','MA','MD','MO','NJ','NM','NY','NC','PA','SC','TN','UT','VA','DC') |
| `in_state_grant_pct` | % of grants to recipients in same state | >= 0.80 (80%+ = hyperlocal) |
| `median_grant` | Median grant size | BETWEEN 10000 AND 200000 |
| `avg_grant_amount` | Average grant size | BETWEEN 10000 AND 200000 |
| `total_giving_5yr` | 5-year total giving | > 0 |
| `unique_recipients_5yr` | Recipient diversity | - |
| `primary_sector` | NTEE sector focus | - |
| `grants_to_orgs` | Makes org grants | = TRUE |
| `preselected_only` | Open to applications | = FALSE or NULL |
| `assets` | Foundation size | - |

### Columns Used from `fact_grants`

| Column | Purpose | Filter Applied |
|--------|---------|----------------|
| `foundation_ein` | Join to foundation | - |
| `recipient_ein` | Exclude existing Horizons funders | NOT IN (17 Horizons affiliate EINs) |
| `purpose_text` | Grant purpose matching | ILIKE any of: education, youth, children, child, student, school, summer, enrichment, after-school, literacy, learning, academic |

### Matching Criteria Summary

1. **Geographic**: Foundation HQ in one of 21 Horizons states
2. **Hyperlocal**: 80%+ of grants stay in-state (`in_state_grant_pct >= 0.80`)
3. **Grant Size**: Median or average grant $10K-$200K
4. **Program Fit**: Has given grants with education/youth keywords
5. **Open**: Accepts applications (`preselected_only = FALSE`)
6. **Not Already Funding**: Excludes 179 foundations already funding Horizons network
