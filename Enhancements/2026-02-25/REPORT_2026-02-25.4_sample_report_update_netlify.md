## Remaining Manual Steps

### Netlify Setup (Alec, in Netlify dashboard)

1. **Import site:** Log into [app.netlify.com](https://app.netlify.com), click "Add new site" > "Import an existing project" > select `akleinman405/thegrantscout-landing` from GitHub
2. **Build settings:** Netlify will auto-detect from `netlify.toml`. Verify: Build command = `npm run build`, Publish directory = `.next`
3. **Custom domain:** Add `thegrantscout.com` in Site settings > Domain management
4. **Update DNS:** Point `thegrantscout.com` to Netlify's nameservers (or add CNAME/A records per Netlify instructions)
5. **SSL:** Netlify provisions Let's Encrypt SSL automatically after DNS propagates
6. **Disable Vercel:** Once Netlify is live and verified, disconnect the project from Vercel dashboard

### DNS Transition Notes

- Vercel and Netlify can briefly coexist (Vercel serves until DNS switches)
- DNS propagation typically takes 1-48 hours
- Test with `curl -I https://thegrantscout.com` to confirm the new host header
