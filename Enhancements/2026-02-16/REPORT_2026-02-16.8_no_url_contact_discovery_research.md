## Executive Summary

95,494 foundations in our database have no website URL. This research investigated how to discover contact information for them through non-web means. The findings are surprisingly optimistic.

**The headline:** We can reach the vast majority of these foundations with data we already have or can acquire cheaply. The "unreachable" segment is small.

**Key findings:**

- **89.6% already have phone numbers** from IRS filings (85,600 of 95,494)
- **4,515 have application contact emails** in `app_contact_email`
- **Reverse phone lookup** (Trestle API) can yield 25,000-35,000 email addresses from existing phone numbers for ~$6,000
- **Address classification** (Smarty) can segment all 95K addresses into residential/commercial/bank for ~$300
- **Bank-managed foundations** (13,242 or 13.9%) have structured contact paths through bank philanthropy desks and portals
- **7,809 viable targets** pass all production filters (active 2022+, assets >= $100K, open to applications, 5+ unique recipients). More than the 4,672 viable with-website foundations.
- Only **261 viable foundations** have zero contact information (no phone, no email) -- 3.3% of the viable target set

**Recommended investment:** ~$10,000-$12,000 across 5 phases to acquire 30,000-50,000 new contact records (blended $0.21-$0.35 per contact).

## Contact Discovery Approaches

### Approach 1: Reverse Phone Lookup (Trestle API)

**What:** Run 85,600 existing phone numbers through Trestle's Reverse Phone API to discover email addresses for foundation officers.

| Attribute | Value |
|-----------|-------|
| Coverage | 85,600 phone numbers (89.6% of no-website foundations) |
| Expected yield | 25,000-35,000 email addresses (40-60% of valid phones have email) |
| Cost | ~$6,000 base + $430 email deliverability = ~$6,430 |
| Pre-filter | Twilio Line Type Intelligence ($685) to identify active numbers first |
| Legal risk | Low (standard identity resolution practice) |
| Implementation | API integration, 1-2 weeks |

**Recommended pre-filter pipeline:** Twilio ($685 for 85,600 numbers) to identify active/valid numbers, then Trestle ($0.07/query) on active numbers only. This reduces Trestle cost by 20-30%.

**Rating: 4.2/5** -- Best ROI tool for our specific data.

### Approach 2: USPS Address Classification (Smarty)

**What:** Classify all 95,494 mailing addresses as residential, commercial, PO Box, or vacant to segment outreach strategy.

| Attribute | Value |
|-----------|-------|
| Coverage | 95,494 addresses (100%) |
| Expected yield | Segmentation data (not direct contacts) |
| Cost | ~$100-$300 for bulk run |
| Legal risk | None |
| Implementation | API batch, 1 day |

**Value:** Address type drives outreach strategy. Residential = family foundation (personal letter + phone). Commercial non-professional = may have staff. Bank/law firm address = contact through professional. This is the cheapest, highest-coverage action available.

**Rating: 4.8/5** -- Universal coverage, near-zero cost.

### Approach 4: Physical Mail Campaign

**What:** Send personalized letters to high-value foundations, followed by phone calls.

| Attribute | Value |
|-----------|-------|
| Coverage | 95,494 (100% have mailing addresses) |
| Expected response rate | 3-5% cold letter, 8-15% with phone follow-up |
| Cost per piece | $0.75-$1.50 (postage + print) |
| Pilot recommendation | 1,000-2,000 highest-scoring foundations |
| Pilot cost | $750-$3,000 |
| Legal risk | None |
| Implementation | 2-3 weeks (design, print, mail, follow-up) |

**Why mail works for small foundations:**
- 70% of recipients find direct mail more personal than digital
- 56% consider it the most trustworthy channel
- 100% delivery to valid addresses (no spam filter)
- Many foundation trustees are older individuals who prefer traditional communication
- Letters sit on desks; emails disappear in inboxes

**Best for:** The 261 viable foundations with zero digital contact info, plus high-value targets where relationship-building matters.

**Rating: 4.0/5** -- Universal coverage, proven effectiveness, moderate cost.

### Approach 5: IRS BMF Extract (ICO Field)

**What:** Download the IRS Exempt Organizations Business Master File and cross-reference the ICO (In Care Of) field against our data.

| Attribute | Value |
|-----------|-------|
| Coverage | 100% of all filers |
| Expected yield | Contact person names; updated addresses |
| Cost | Free |
| Legal risk | None (public data) |
| Implementation | CSV download + SQL import, 1 day |

The ICO field names the person to whom IRS correspondence is directed. This may differ from officers on the 990-PF and provides another contact person name for outreach. Also provides the most current address on file.

**Rating: 4.0/5** -- Free, universal, but names only (no email/phone beyond what we have).

### Approach 6: Candid Search Subscription

**What:** Subscribe to Candid Search ($100/mo after January 2026 price drop) and cross-reference foundation profiles for contact data.

| Attribute | Value |

| Cost | $100/month ($1,200/year) |
| Enriched data for small foundations | Low (5-15% of our no-website set) |

| Legal risk | None |

Candid is the gold standard for larger/professional foundations but has limited enriched data for small family foundations. The January 2026 price drop (from $299/mo to $100/mo) makes it worth subscribing as a supplementary source.

**Rating: 3.8/5** -- Good for professional foundations, weak for family foundations.

## Bank-Managed Foundation Playbook

13,242 no-website foundations (13.9%) are bank/trust-managed. These have the most structured contact paths of any segment.

### Bank Portal Summary

| Bank | Our Foundations | On Portal | Portal % | Portal URL |
|------|------:|------:|-----:|-----------|
| JPMorgan | 579 | ~115 | 20% | jpmorgan.com/private-bank/foundations/online-applications/search |
| Bank of America | 2,319 | ~160 | 7% | bankofamerica.com/philanthropic/search-for-grants |
| Wells Fargo | 2,710 | ~200 | 7% | wellsfargo.com/private-foundations |
| PNC | 936 | ~200 | 21% | pnc.com/.../charitable-trusts.html |
| Truist | 548 | 6 | 1% | truist.com/trusteed-foundations/funds |
| BNY Mellon | 790 | 0 | 0% | None |
| US Bank | 972 | 0 | 0% | None |
| Northern Trust | 163 | 0 | 0% | None |

### Recommended Actions

1. **Scrape JPMorgan Foundation Finder.** 115 foundations with deadlines are embedded in page JavaScript. Cross-reference with our 579 JPMorgan foundations.

2. **Build `bank_trustee_contacts` enrichment table.** For each bank: philanthropy desk phone, email, portal URL. Store per-foundation: `portal_listed` boolean, `portal_url`, `trustee_bank`.

3. **Include bank contact info in client reports.** When a scored foundation is bank-managed, the report should include: "This foundation is managed by [Bank]. Apply at [portal URL] or call [phone]."

4. **Monitor Northern Trust + Foundation Source partnership.** Northern Trust recently partnered with Foundation Source (fsrequests.com), which could make their 163+ foundations accessible through Foundation Source's platform.

### How Grantseekers Contact Bank-Managed Foundations (Standard Workflow)

1. Identify the trustee bank via 990-PF officer listings
2. Check the bank's public portal for the specific foundation
3. If not on portal, call the bank's philanthropy desk and ask by foundation name
4. Submit LOI or application per the bank's instructions

This is well-established industry practice. Grant databases (Instrumentl, GrantStation, Candid) all recognize bank-trusteed foundations as contactable.

## A/B Testing Plan

### Test 1: Reverse Phone Email Yield Rate

**Hypothesis:** Trestle reverse phone lookup will yield email addresses for 40-60% of foundations with valid phone numbers.

| Parameter | Value |
|-----------|-------|
| Sample size | 1,000 phone numbers (stratified by asset tier) |
| Control | N/A (testing yield rate, not response) |
| Success metric | Email found rate per phone lookup |
| Cost ceiling | $70 (1,000 x $0.07) |
| Decision threshold | If yield < 25%, reconsider Phase 2 budget. If > 50%, accelerate. |
| Duration | 1 day |

**Stratification:**
- 250 from < $100K tier
- 250 from $100K-$1M tier
- 250 from $1M-$5M tier
- 250 from $5M+ tier

### Test 2: Mail Campaign Response Rate

**Hypothesis:** Personalized letters to residential-address foundations will generate 5-8% response rate, rising to 10-15% with phone follow-up.

| Parameter | Value |
|-----------|-------|
| Sample A | 500 letters with phone follow-up (residential addresses, $1M+ assets) |
| Sample B | 500 letters only, no phone follow-up (same criteria) |
| Success metric | Response rate (reply via any channel within 30 days) |
| Cost ceiling | $1,500 ($750 per arm) + staff time for calls |
| Decision threshold | If combined rate < 3%, deprioritize mail. If > 8%, scale to 5,000. |
| Duration | 6 weeks (mail + 30-day response window) |

### Test 3: Apollo.io Enrichment Hit Rate

**Hypothesis:** Apollo.io will find email addresses for 10-20% of foundation officers given name + organization.

| Parameter | Value |
|-----------|-------|
| Sample size | 1,000 officer records (name + foundation name + state) |
| Success metric | Verified email found rate |
