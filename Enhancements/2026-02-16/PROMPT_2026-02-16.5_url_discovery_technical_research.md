# PROMPT: Technical Approaches for Foundation URL Discovery

**Date:** 2026-02-16
**Sequence:** .5
**Type:** Verbal instruction in Claude Code CLI

---

## Context/Situation

TheGrantScout has 143,184 foundations in its database but only 14,107 (9.9%) with validated URLs. We need to find more URLs to support the email pipeline. Existing tools include a DuckDuckGo scraper (83.7% hit rate in Dec 2025, 25% in Feb 2026), a pattern-guessing script, and IRS-filed URLs.

## Tasks

Research 7 categories of technical approaches for discovering foundation website URLs:

1. Data enrichment APIs (Clearbit, Hunter.io, ZoomInfo, Apollo.io, Snov.io, Lusha, RocketReach, FullContact)
2. Search engine APIs (Google Custom Search, Bing, SerpAPI, Serper.dev, DuckDuckGo, Brave)
3. Domain discovery techniques (DNS lookup, domain pattern generation, WHOIS, Certificate Transparency)
4. Archive and historical data (Wayback Machine, Common Crawl)
5. Social media and professional networks (LinkedIn, Facebook, Twitter)
6. Web scraping infrastructure (proxies, managed services, headless browsers)
7. Open source tools (theHarvester, recon-ng, nonprofit data tools)

For each approach, provide: what it does, foundation coverage/accuracy, cost, effort, rate limits, legal risk, and verdict.

End with a ranked top 5 and implementation roadmap.
