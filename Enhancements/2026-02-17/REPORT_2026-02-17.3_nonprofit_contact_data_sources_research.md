## Executive Summary

After thorough research across 15+ data sources, the key finding is stark: **no free public source provides bulk nonprofit email addresses**. Email addresses are simply not part of any government filing requirement (IRS 990, state registries), and the aggregators that do collect them (Candid, Cause IQ) lock them behind paid tiers.

**What IS freely available in bulk:**
- Organization phone numbers (from IRS 990 XML -- we already have 2.9M)
- Officer/director names and titles (from IRS 990 Part VII -- we already have 26.3M records across 792K orgs)
- Mission descriptions and program info (from IRS 990 -- we already have 2.96M)
- Websites/URLs (from IRS 990 -- we already have 2.5M)
- Organization addresses (from IRS EO BMF, 990 XML, state registries)
- "In Care Of" contact name (from IRS EO BMF -- 1.9M records)

**What requires payment or scraping:**
- Contact email addresses (Candid Premier API, Cause IQ, or website scraping)
- Verified current contact person names with emails (Candid Premier only)

**Bottom line:** The path to nonprofit emails at scale is either (a) Candid Premier API ($$$, enterprise pricing), (b) scraping nonprofit websites using the 578K URLs we already have, or (c) commercial email list vendors (~$0.01-0.10/record).

## Source Details

### 1. ProPublica Nonprofit Explorer API

**URL:** https://projects.propublica.org/nonprofits/api

**Endpoints:**
- `GET /search.json` -- search by name, state, NTEE code, etc.
- `GET /organizations/:ein.json` -- lookup by EIN

**Data Fields Available:**
- EIN, name, address, city, state, zip
- Subsection code (501(c) type), NTEE code
- Filing data: total revenue, expenses, assets, liabilities
- 40-120 form-specific fields per filing
- Links to GuideStar and NCCS profiles

**Contact Data:** NONE. No email, no phone, no officer names, no mission description in the API response. The API is a financial data wrapper, not a contact directory.

**Bulk Download:** No bulk download. API only.
**Rate Limits:** Not publicly documented. PDF download links are rate limited.
**Records:** ~1.8M organizations
**TOS:** No explicit prohibition on bulk use, but no bulk endpoint exists.
**Freshness:** Updated periodically as IRS releases new data.

**Verdict:** Useless for contact enrichment. We already have superior data in our own database.

### 2. IRS Tax Exempt Organization Search (TEOS) & EO BMF

**URL:** https://www.irs.gov/charities-non-profits/tax-exempt-organization-search-bulk-data-downloads

**Bulk Downloads Available:**

| File | Format | Key Fields | Email? |
|------|--------|------------|--------|
| EO BMF Extract | CSV (by region) | EIN, name, ICO name, address, state, NTEE, subsection, assets, income, ruling date | NO |
| Publication 78 | Pipe-delimited | EIN, name, city, state, deductibility code | NO |
| Auto Revocation List | Pipe-delimited | EIN, name, address, revocation date | NO |
| Form 990-N (e-Postcard) | Pipe-delimited | EIN, name, address, tax period, gross receipts | NO |
| Form 990 Series XML | XML (monthly ZIPs) | Full 990 filing data: financials, officers (Part VII), mission, phone, programs | NO email |

**Critical Detail on EO BMF:**
- 1,924,822 records (as of Dec 2025)
- The "ICO" (In Care Of) field contains a contact person name -- this is the ONE contact name field
- NO email, NO phone number in the BMF extract
- Updated monthly

**Critical Detail on 990 XML:**
- Phone number: YES (organization phone number is a required field on the 990)
- Officer/director names: YES (Part VII lists names, titles, compensation)
- Mission description: YES (Part III)
- Program descriptions: YES (Part III)
- Email: The XML schema includes `EmailAddressTxt` but the IRS **strips/redacts** it from public downloads
- Website URL: YES

**We already have all of this parsed.** Our `nonprofit_returns` table has 2.96M filings with phone (98.2% populated), website (85.5% populated), and mission. Our `officers` table has 26.3M records across 792K distinct orgs.

**Verdict:** The IRS 990 XML is the single best FREE source for nonprofit data. We already have it fully ingested. The gap is emails, which the IRS does not disclose.

### 4. Candid / GuideStar

**URL:** https://developer.candid.org/

**API Tiers:**

| Tier | Key Data | Contact Fields | Cost |
|------|----------|----------------|------|
| **Free (GuideStar search)** | Name, EIN, state, NTEE, basic financials | NONE | Free (limited searches/month) |
| **GuideStar Pro** | 40+ search filters, 50+ data points per org, 5K exports/month | NONE (no email) | $396/yr |
| **Essentials API** | Search with filters, basic org info | NONE | Enterprise pricing (contact sales) |
| **Premier API** | Full profiles: financials, people, DEI, compliance | **YES: contact_name, contact_title, contact_email, contact_phone, telephone_numbers[]** | Enterprise pricing ($$$$) |

**Premier API Confirmed Fields:**
- `contact_name` -- primary contact person
- `contact_title` -- their title
- `contact_email` -- their email address
- `contact_phone` -- their phone
- `telephone_numbers[]` -- array with `telephone_number` and `telephone_type`
- Full mission, programs, address, financials, people/leadership data

**Coverage:** 1.6M+ nonprofits
**Rate Limits:** Defined but not publicly documented
**Freshness:** Continuously updated as nonprofits update their profiles
**TOS:** API usage governed by developer agreement

**Verdict:** Candid Premier API is the ONLY aggregated source with verified nonprofit email addresses at scale. However, pricing is enterprise-level (likely $5K-25K+/year based on industry norms) and not self-serve. This is the gold standard but expensive.

- NO email, NO phone, NO officer names in the free tier
- Paid tiers planned but not yet released
- Coverage: 1.8M organizations (basic data), ~225K rated

**Profile Pages (web):**

- Email addresses are NOT consistently displayed
- No bulk download available

**TOS:** Explicitly requires developer registration and approval

**Verdict:** Useful for ratings data but NOT for contact enrichment. The GraphQL API is too limited, and even full profiles rarely include email.

## Recommended Acquisition Strategy

### Tier 1: Free / Near-Free (Do Now)

1. **Website Scraping Pipeline** -- We have 578K nonprofit website URLs. Build a scraper that visits each site's `/contact`, `/about`, `/staff` pages and extracts email addresses. Tools like Apify, Outscraper (free tier), or custom Python (requests + BeautifulSoup) can do this. Expected yield: 30-50% of sites will have a visible email address = **170K-290K emails**.

2. **IRS BMF "ICO" Name Extraction** -- Download the latest EO BMF CSV (1.92M records). The ICO (In Care Of) field contains a contact person name for correspondence. Cross-reference with our `officers` table to enrich.

3. **GivingTuesday Personnel Datamarts** -- Free, open API. Download their Personnel datamart to get officer names in a clean format. May complement our existing `officers` table.

### Tier 2: Low Cost ($500-5,000)

4. **Cause IQ Pro** ($4,488/yr) -- Unlimited exports with officer names, emails, phone numbers across the full IRS corpus. Best bang-for-buck if we can budget it. Would fill the email gap for a large portion of the 673K NP prospects.

5. **Commercial Email Lists** -- Vendors like NonprofitLists.com offer 140K org emails for a one-time fee (likely $500-2,000). Quality is unknown and probably mixed, but could bootstrap the pipeline.

### Tier 3: Premium ($5,000+)

6. **Candid Premier API** -- The gold standard. Verified contact_name + contact_email + contact_phone for 1.6M nonprofits. Enterprise pricing (likely $10K-25K/year). Worth it if email outreach becomes a core revenue channel.

### Priority Recommendation

**Start with #1 (website scraping).** We already have 578K URLs. A well-built scraper running over a few days could yield 200K+ emails at zero marginal cost. This is the highest-ROI action.

**Then evaluate #4 (Cause IQ Pro)** as the backup/supplement. At $4,488/yr with unlimited exports, it would fill most remaining gaps.
