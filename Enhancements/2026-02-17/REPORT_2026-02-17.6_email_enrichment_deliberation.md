## Executive Summary

Four research agents independently investigated every angle for acquiring nonprofit email addresses at near-zero cost. Their conclusions converge strongly:

**There is no single free source of bulk nonprofit emails.** Email is the one contact field that no government database, no free API, and no public directory exposes at scale. The IRS collects emails but strips them from public releases. SAM.gov has them but restricts access to federal accounts. Candid/GuideStar has them but charges enterprise pricing.

**The path forward is a build-it-ourselves pipeline** that combines three strategies in sequence:

| Strategy | Expected Emails | Cost | Time |
|----------|----------------|------|------|
| Website scraping (292K validated URLs) | 120K-160K | $0-15 | 3-4 days |
| Generic email generation + verification (info@, contact@) | 100K-170K | $0-200 | 3-8 days |
| Pattern inference from IRS officer names | 30K-80K | $0 | 5-10 days |
| **Total (with overlap/dedup)** | **200K-300K** | **$5-210** | **2-4 weeks** |

Before starting any of that, there is one critical verification: **check whether the IRS 990 XML public download actually contains the `EMailAddressTxt` field.** If it does, this could yield 200K-400K emails with just a parser update. Agent 4 identified this as the single highest-leverage action. Our 990-PF parser already tries to extract this field and finds 0 rows, which suggests the IRS strips it, but we should verify with 990/990-EZ files too.

**For names and org focus area:** We already have these. 623K nonprofits have officer names (from IRS Part VII), all 673K have mission descriptions, and all have NTEE codes. The gap is purely email.

## What the 4 Agents Found

### Agent 1: Free Directories & Public Data Sources

Investigated 15+ data sources including ProPublica API, IRS TEOS/BMF, state charity registries, Candid/GuideStar, Charity Navigator, CharityAPI.org, NCCS, GivingTuesday, Open990, Cause IQ, GreatNonprofits, Idealist, and commercial list vendors.

**Key finding:** No free public source provides bulk nonprofit email addresses. The closest options:
- **Candid Premier API** has verified emails but costs $5K-25K+/year (enterprise pricing)
- **Cause IQ Pro** ($4,488/yr) offers unlimited exports with emails
- **State registries** don't include email (even California's CSV downloads)
- **Commercial list vendors** ($0.01-0.10/record) offer questionable quality

**Recommendation:** Website scraping is the only viable free path to email at scale.

### Agent 2: Web Scraping at Scale

Deep-dived into scraping tools, infrastructure, page targeting, legal considerations, and timeline estimates.

**Key finding:** Two-phase scraper (aiohttp + Playwright) can process 292K sites in 3-4 days for $0-15.
- Phase 1: HTTP-only (85% of sites): 100-500 pages/sec with aiohttp
- Phase 2: JS rendering (15% of sites): 5-20 pages/sec with Playwright
- Page targeting: /contact (60-70% yield), /about (30-40%), /staff (40-50%)
- Expected yield: 120K-160K emails (40-55% of scraped sites)
- Legal: Green light under hiQ v. LinkedIn precedent

**Recommendation:** Build custom Python scraper, run on MacBook Pro or cheap EC2 instance.

### Agent 3: Email Discovery Tools & Methods

Researched email verification tools, generic email generation, pattern inference, DNS approaches, OSINT tools, and social media cross-referencing.

**Key finding:** 5-phase pipeline combining multiple free methods yields 200K-300K emails for $5-210.
- Phase 1: DNS MX pre-filter (eliminates 13% of domains)
- Phase 2: Generic role emails (info@, contact@) + verification: 100K-170K
- Phase 3: Website scraping: 50K-120K additional
- Phase 4: Pattern inference from officer names: 30K-80K
- Phase 5: Free API tiers (Apollo.io 10K/mo): ongoing enrichment

**Key infrastructure need:** VPS with port 25 open ($5-10/mo) for self-hosted Reacher email verification.

**Recommendation:** Multi-phase pipeline starting with DNS filter, then generic emails, then scraping, then pattern inference.

### Agent 4: Database Mining & IRS Data

Examined IRS 990 XML fields in detail, 990-N e-Postcard data, SAM.gov, state registries, USASpending.gov, and our existing parsers.

**Key finding:** The `EMailAddressTxt` field exists in the IRS 990 XML schema but is likely stripped from public releases. Our 990-PF parser already tries to extract it (line 137 of parse_990pf.py) and finds 0 rows. The 990/990-EZ parsers don't even try.

**Critical action:** Download 5-10 actual 990/990-EZ XML files from IRS TEOS and verify whether the email field is populated or stripped. If populated: 200K-400K emails with a simple parser update.

Other findings:
- 990-N e-Postcard: IRS collects filer email but strips it from public download (all 26 fields documented, no email)
- SAM.gov: Has emails for 80K-120K nonprofits but classified FOUO (federal accounts only)
- SAM.gov public extract: Has POC names/titles but no email/phone
- State registries: California only state with CSV downloads, no email field

**Recommendation:** Verify 990 XML email field first (1 hour). If empty, proceed with scraping pipeline.

### Phase 4: Pattern Inference (Days 12-20)

- For remaining domains with officer names but no email
- Generate 8 pattern permutations per officer (first.last@, first@, flast@, etc.)

**Expected output:** 30K-80K personal emails

## Critical First Step

**Before building anything, spend 1 hour verifying the IRS 990 XML email field.**

Agent 4 identified that the IRS 990 XML schema has `/Return/ReturnHeader/Filer/EMailAddressTxt` for all form types. Our 990-PF parser extracts it but finds 0 values. Our 990/990-EZ parsers don't even try.

**Steps:**
1. Download 5-10 recent 990 and 990-EZ XML files from the IRS TEOS bulk download
2. Open them and search for `EMailAddressTxt` or `EmailAddress`
3. If populated: update `parse_990.py` and `parse_990ez.py` to extract it, re-process all XMLs
4. If empty (likely): document and move on to the scraping pipeline

**Why this matters:** If the field is populated, it's a 200K-400K email windfall with ~2 hours of work. The expected-value of checking far exceeds the 1-hour investment.
