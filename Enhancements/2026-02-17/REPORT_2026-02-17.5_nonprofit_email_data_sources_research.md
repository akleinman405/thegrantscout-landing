## Executive Summary

We currently have email addresses for **15,088 unique EINs** out of 816,564 total organizations (673,380 nonprofits + 143,184 foundations) -- a 1.8% email coverage rate. After exhaustive research into 9 data source categories, the findings are sobering but actionable:

**The two "goldmine" sources (990-N and SAM.gov) are NOT what they seemed:**
- **990-N e-Postcard:** The IRS collects the filer's email during submission, but the bulk download does NOT include the email field. Only 26 fields are published; email is deliberately excluded.
- **SAM.gov:** Email addresses exist but are classified as **FOUO (For Official Use Only)** -- accessible only to federal system accounts, not the general public.

**The biggest realistic opportunity is data we already have but are not fully extracting:**
- The IRS 990/990-EZ XML e-files contain an `EMailAddressTxt` field in the Return Header Filer section, but **our 990/990-EZ parser does not extract it**. This could yield emails for up to ~400K+ nonprofits.
- The 990-PF parser does attempt to extract `email_address_txt` from this field, but it shows **0 populated rows** -- meaning either the xpath is wrong or the IRS strips it from public releases.

**Realistic new email yield estimate across all sources: 50,000-150,000 incremental emails** over current baseline, with the IRS 990 XML header email being the single largest opportunity IF the data exists in public releases.

## Source 1: IRS 990 XML E-File Data {#source-1-irs-990-xml}

### Email Fields in the IRS 990 XML Schema

Based on examination of our XML parsers and the IRS schema documentation:

**Return Header (all 990 forms):**
- **`/Return/ReturnHeader/Filer/EMailAddressTxt`** -- Email address of the organization for IRS correspondence
- This field exists in the schema for 990, 990-EZ, AND 990-PF
- Our 990-PF parser attempts to extract it but finds 0 records populated
- Our 990/990-EZ parsers DO NOT attempt to extract it at all

**990-PF Part XV Application Submission Info:**
- **`/Return/ReturnData/IRS990PF/SupplementaryInformationGrp/ApplicationSubmissionInfoGrp/RecipientEmailAddressTxt`** -- Grant application contact email
- This IS populated: 35,807 returns, 8,662 unique EINs with valid email addresses
- Many entries are junk ("N/A", "NONE", "NOT APPLICABLE") -- about 3,800 rows
- About 31,000 rows contain valid email addresses

**Business Officer Group (Return Header):**
- **`/Return/ReturnHeader/BusinessOfficerGrp/PersonNm`** -- Officer name
- **`/Return/ReturnHeader/BusinessOfficerGrp/PersonTitleTxt`** -- Officer title
- **`/Return/ReturnHeader/BusinessOfficerGrp/PhoneNum`** -- Officer phone
- **No email field in BusinessOfficerGrp** -- confirmed from schema samples

**Preparer Person Group:**
- **`/Return/ReturnHeader/PreparerPersonGrp/PreparerPersonNm`** -- Preparer name
- **`/Return/ReturnHeader/PreparerPersonGrp/PhoneNum`** -- Preparer phone
- **`/Return/ReturnHeader/PreparerPersonGrp/PTIN`** -- Preparer Tax ID
- **No email field for preparer** -- phone and PTIN only

### Critical Finding: Email May Be Stripped from Public Releases

The IRS 990 XML schema DEFINES `EMailAddressTxt` in the Filer section, and tax preparation software collects it. However, it appears the IRS may strip this field from the public XML releases on TEOS/AWS. Evidence:
- Our 990-PF parser explicitly looks for it and finds 0 records
- No researchers in the nonprofit data community reference extracting email from the header
- The IRS data is reviewed for PII before public release

### Assessment

| Criterion | Value |
|---|---|
| Contains email? | Schema: YES. Public data: LIKELY NO (stripped) |
| Contains contact names? | YES (officer, preparer) |
| Contains mission/focus? | YES (mission, NTEE, programs) |
| Coverage | ~1.7M e-filed 990s on TEOS (2017+) |
| Access method | Bulk XML download from IRS TEOS |
| Freshness | Updated monthly |
| Legal restrictions | Public record, no restrictions |

## Source 2: IRS 990-N E-Postcard {#source-2-irs-990-n}

### What the 990-N Is

Form 990-N (e-Postcard) is filed by small tax-exempt organizations with gross receipts normally <= $50,000. It is a very simple electronic notice -- just 8 items. The filer provides their email address to file.

### The 990-N Bulk Download Fields (All 26)

Per the NCCS/Urban Institute dataset documentation and the Nonprofit Open Data Collective:

1. EIN
2. Tax Year
3. Organization Name
4. Gross Receipts Under $25,000
5. Terminated
6. Tax Period Begin Date
7. Tax Period End Date
8. Website URL
9. Officer Name
10. Officer Address Line 1
11. Officer Address Line 2
12. Officer Address City
13. Officer Address Province
14. Officer Address State
15. Officer Address Postal Code
16. Officer Address Country
17. Organization Address Line 1
18. Organization Address Line 2
19. Organization Address City
20. Organization Address Province
21. Organization Address State
22. Organization Address Postal Code
23. Organization Address Country
24. Doing Business As Name 1
25. Doing Business As Name 2
26. Doing Business As Name 3

**EMAIL IS NOT INCLUDED.** The IRS collects the filer's email during submission but does NOT release it in the public bulk download. This is a deliberate PII exclusion.

### What IS Useful

- **Officer Name**: Useful for personalizing outreach (not available in 990/990-EZ XML for these small orgs since they only file 990-N)
- **Website URL**: Useful for web scraping to find email
- **~460K unique organizations** with recent filings, ~1.3M total historical filings

| Legal restrictions | None |

## Source 3: IRS Bulk Data Downloads (BMF, Pub 78, TEOS) {#source-3-irs-bulk}

### Exempt Organizations Business Master File (EO BMF)

The EO BMF is the IRS's cumulative registry of all tax-exempt organizations. Fields include:

- EIN, Name, Address (street, city, state, zip)
- ICO (In Care Of name)
- Subsection code, NTEE code, ruling date
- Group exemption number
- Activity codes, foundation type code
- Asset amount, income amount, revenue amount
- Filing requirement code, accounting period

**NO email, NO phone, NO website.** The BMF is purely organizational/classificatory data.

### Publication 78 (Eligible Donee Organizations)

Pipe-delimited list of organizations eligible to receive tax-deductible contributions. Fields:
- EIN, Name, City, State, Country, Deductibility Status

**NO contact information at all.** Purely an eligibility list.

### TEOS Data Dictionaries

The IRS provides data dictionaries for the TEOS bulk data, but based on review:
- The 990/990-EZ/990-PF XML downloads contain whatever the original XML contains (including email IF the IRS does not strip it)
- The 990-N download specifically excludes email
- No supplemental contact databases are published through TEOS

### Assessment

| Criterion | Value |
|---|---|
| Contains email? | NO (BMF, Pub 78); MAYBE (990 XML -- needs verification) |
| Contains contact names? | ICO name in BMF only |
| Coverage | 1.8M+ organizations in BMF |
| Access | Free bulk download |
| Freshness | Monthly updates |

### Estimated Email Yield: 0 (from BMF/Pub78 specifically)

## Source 4: SAM.gov {#source-4-sam-gov}

### What SAM.gov Contains

The System for Award Management (SAM.gov) is where organizations register to do business with the federal government. Any nonprofit that receives federal grants, contracts, or cooperative agreements must register. Registration includes:

- Organization name, address, UEI (Unique Entity Identifier)
- Business type (including "Nonprofit Organization", "501(c)(3)")
- NAICS codes, PSC codes
- **Government Business POC** (name, title, phone, fax, email)
- **Electronic Business POC** (name, title, phone, fax, email)
- **Alternate Government Business POC** (name, title, phone, fax, email)

### The FOUO Problem

SAM.gov entity data comes in three sensitivity levels:

| Level | What's Included | Access |
|---|---|---|
| **Public** | Name, address, UEI, business type, NAICS, PSC, POC names and titles | Free with API key |
| **FOUO (CUI)** | All public data PLUS **email, phone, fax** for all POCs | **Federal system accounts only** |
| **Sensitive** | All FOUO data PLUS banking info, SSN/TIN | Federal system accounts with elevated permissions |

**Email addresses are classified as FOUO/CUI.** They are NOT available in the public extract. You need a federal government system account with "Read FOUO" permission.

### API Rate Limits

| User Type | Requests/Day |
|---|---|
| Non-federal (no role) | 10 |
| Non-federal (with role) | 1,000 |
| Federal individual | 1,000 |
| Federal system account | 10,000 |

### How Many Nonprofits Are in SAM.gov?

Exact statistics are not published, but estimates based on federal grant data:
- ~80,000-120,000 nonprofits likely have active SAM.gov registrations
- This represents orgs that receive federal grants/contracts
- A significant overlap with our existing database, since these tend to be larger, established nonprofits

### Public Extract Contents (What We CAN Get)

From the SAM Entity Management Public V2 Extract Layout:
- Entity name, address, country, congressional district
- UEI and DUNS number
- Business types (including nonprofit designations)
- Registration dates, expiration, purpose of registration
- **POC names and titles** (but NOT email/phone)
- NAICS and PSC codes
- Entity structure, state of incorporation

### Assessment

| Criterion | Value |
|---|---|
| Contains email? | YES -- but **FOUO only**, not in public extract |
| Contains contact names? | YES -- POC names/titles in public extract |
| Contains mission/focus? | NAICS codes only (no free-text mission) |
| Coverage | Est. 80K-120K nonprofits |
| Access | Public: Free API + bulk download. FOUO: Federal accounts only |
| Freshness | Monthly bulk extracts, real-time API |
| Legal restrictions | FOUO data restricted to authorized federal users |

### Estimated Email Yield

- **From public data**: 0 emails, but ~80K-120K POC names that could be used with email-finding tools
- **If you could get FOUO access**: 80K-120K emails (but legally restricted)
- **Realistic path**: Use public POC names + org name to find emails via other means
