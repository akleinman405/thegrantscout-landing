## Project 1: Email Infrastructure Setup

**Goal:** Set up email sending infrastructure for cold email campaigns across thegrantscout.com and filteredmessaging.com.

### What Was Built
- Migrated thegrantscout.com DNS to Cloudflare
- Created **20 email subdomains** (10 per domain) for sender rotation:
  - `ai`, `connect`, `contact`, `email`, `hello`, `info`, `insights`, `mail`, `outreach`, `team` @ thegrantscout.com
  - Same + `partner` @ filteredmessaging.com
- Set up full email authentication for each subdomain:
  - **MX records** pointing to Google Workspace
  - **SPF records** authorizing Google's servers
  - **DMARC** in monitor mode (p=none)
  - **DKIM** keys generated and DNS records added
- Added all 14 new domain aliases in Google Workspace Admin
- Verified 88/88 DNS record checks passed (100%)

### Why 10 Subdomains (Not 30)
Research showed 30 subdomains on one root domain is a red flag for ESPs. 10 per domain with 15 emails/day each = 150/day capacity per domain.
