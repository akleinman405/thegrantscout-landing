## Project 1: Gmail Filter Setup

**Goal:** Set up Gmail filters to automatically sort bounced emails, responses, and auto-replies from two email campaigns (TheGrantScout and Filtered Messaging).

### What Was Built
- Created **5 Gmail filters** that auto-sort campaign email:
  1. **All Bounces** — from mailer-daemon/postmaster → "Bounced Emails" label, skip inbox
  2. **TGS Auto-Replies** — OOO/vacation from TGS subdomains → label, skip inbox, mark read
  3. **TGS Responses** — Real replies from TGS subdomains → label, keep in inbox
  4. **FM Auto-Replies** — OOO from FM subdomains → label, skip inbox, mark read
  5. **FM Responses** — Real replies from FM subdomains → label, keep in inbox
- Built automated test script (`test_gmail_filters.py`) that verifies all 5 filters
- All 5 filters passed verification after 4 iterations to fix edge cases

### Key Discoveries
- `deliveredto:` doesn't work for intra-account Gmail aliases — used `to:` instead
- Gmail doesn't index bounce notification bodies (multipart/report MIME) — used `from:mailer-daemon` catch-all
- Gmail label paths must exactly match existing hierarchy to nest correctly

## Project 2: Email Sourcing Pipeline Research

**Goal:** Document exactly how TGS sourced email addresses for the Nov-Dec 2025 email campaign.

### Key Findings
The TGS nonprofit email campaign used a 3-step pipeline:

1. **SQL Export** (`export_ideal_prospects.py`) — Queries IRS 990 data to identify ideal nonprofit prospects with websites and contact names. Selection: 3-4 foundation funders, revenue <$2.5M, has website, recent grants.

2. **Website Scraping** (`email_scraper.py`) — Async scraper hits prospect websites to extract email addresses. Tries homepage first, then follows contact page links. Scores emails by domain match, filters false positives (50+ exclude patterns). Achieves ~15-30% yield.

3. **Campaign Sending** (`campaign_manager.py`) — Multi-sender system with subdomain rotation, business-hours pacing (Mon-Fri 9am-7pm), 15 emails/day per sender. Bounce and reply checking via Gmail API.

### Campaign Results (Nov 30, 2025 snapshot)
| Metric | Value |
|--------|-------|
| Total Sent | 1,844 |
| Delivered | 1,731 (93.9%) |
| Bounced | 113 (6.1%) |
| Replied | 4 (0.23%) |

### Key Gap Identified
TGS has no nightly automation (unlike Filtered Messaging's Psychology Today scraper). The prospect pool is static IRS data updated annually. The scraper was run in ad-hoc batches.
