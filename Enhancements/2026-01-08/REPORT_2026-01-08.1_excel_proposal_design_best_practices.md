## Executive Summary

Creating professional Excel workbooks for B2B sales proposals requires attention to visual design, consistent branding, and careful formatting. The key principles are:

1. **Limit colors to 2-3** - Use a primary brand color, a neutral (gray/black), and one accent
2. **Use sans-serif fonts** - Calibri, Arial, or Helvetica for maximum readability
3. **Remove visual clutter** - Gridlines, excessive borders, and unnecessary axis lines
4. **Format data as Excel Tables** - For automatic styling, filtering, and banded rows
5. **Include professional branding** - Company logo in headers, consistent color scheme throughout
6. **Optimize for print** - Custom margins, page breaks, and centered content

### How to Create a Custom Theme

1. Go to **Page Layout** tab > **Colors** dropdown
2. Click **Customize Colors** at the bottom
3. Define your custom palette:
   - **Dark 1:** Black or dark gray (text)
   - **Light 1:** White or very light gray (background)
   - **Accent 1-6:** Your brand colors
4. Save with a descriptive name (e.g., "TheGrantScout Proposal")

## Typography and Fonts

### Recommended Fonts

| Element | Font | Size | Style |
|---------|------|------|-------|
| **Titles** | Calibri | 16-18pt | Bold |
| **Section Headers** | Calibri | 14pt | Bold |
| **Subheaders** | Calibri | 12pt | Bold |
| **Body/Data** | Calibri | 10-11pt | Regular |
| **Table Headers** | Calibri | 10-11pt | Bold, White text |
| **Footnotes** | Calibri | 9pt | Italic |

### Font Best Practices

- **Use sans-serif fonts only**: Calibri, Helvetica, Arial, or Segoe UI
- **Maintain hierarchy**: Headers > Subheaders > Body text (larger to smaller)
- **Text alignment**:
  - **Left-align** text and labels
  - **Right-align** numbers and currency
  - **Center** column headers only
- **Never use Comic Sans, Papyrus, or decorative fonts**

### Number Formatting

| Data Type | Format Code | Example |
|-----------|-------------|---------|
| Currency | `"$"#,##0` | $125,000 |
| Large numbers | `#,##0` | 1,250,000 |
| Percentages | `0%` or `0.0%` | 45% or 45.5% |
| Millions | `#,##0,,"M"` | 5M |
| Thousands | `#,##0,"K"` | 500K |

## Data Table Formatting

### Creating Professional Tables

1. **Convert to Excel Table** (Ctrl+T / Cmd+T)
   - Automatically adds banded rows
   - Enables filter/sort buttons
   - Expands formatting with new data

2. **Apply a Professional Table Style**
   - Use **TableStyleMedium2** or similar
   - Avoid overly colorful styles
   - Enable "Banded Rows" for readability

### Table Design Standards

| Element | Specification |
|---------|---------------|
| **Blank row 1** | Leave row 1 empty (data starts at row 2) |
| **Blank column A** | Leave column A empty (data starts at column B) |
| **Zoom level** | Set to 140% for better readability |
| **Header row background** | Dark blue (`#2F5496`) |
| **Header row text** | White, Bold |
| **Border - outside** | Medium weight, dark gray |
| **Border - header bottom** | Medium weight, darker than inside borders |
| **Border - inside** | Thin weight, light gray |
| **Text wrap** | OFF (use column width instead) |
| **Freeze panes** | Freeze header row and first column |

### Conditional Formatting for Impact

| Use Case | Formatting Rule |
|----------|-----------------|
| Highlight top values | Data bars (blue gradient) |
| Show positive/negative | Green text for positive, red for negative |
| Flag attention items | Yellow or orange cell fill |
| Rank performance | Icon sets (arrows or traffic lights) |

## Professional Chart Design

### Chart Type Selection

| Data Story | Best Chart Type |
|------------|-----------------|
| Comparison across categories | Clustered Bar or Column |
| Trends over time | Line chart |
| Part of whole | Pie (limit to 5-6 slices) or Stacked Bar |
| Portfolio/variance analysis | Bar chart |
| Cash flow analysis | Waterfall chart |
| Performance vs. target | Combo chart (bars + line) |
| KPI at a glance | Gauge chart |

### Removing Visual Clutter

Remove these elements for a cleaner, professional look:

1. **Gridlines**: Chart Elements (+) > Uncheck Gridlines
2. **Axis lines**: Format Axis > Fill & Line > No Line
3. **Chart border**: Format Chart Area > Border > No Line
4. **Plot area background**: Format Plot Area > Fill > No Fill

### Chart Formatting Checklist

- [ ] Remove all gridlines
- [ ] Remove axis lines (or make light gray)
- [ ] Remove chart border
- [ ] Add direct data labels (reduce need for legends)
- [ ] Use brand colors only
- [ ] Change large numbers to display units (M, K)
- [ ] Remove default chart title (add a better one above chart)
- [ ] Align chart edges to grid

### Chart Color Rules

- Use **one primary color** for main data series
- Use **gray** for comparison/baseline data
- Use **accent color** only to highlight key data points
- Avoid gradients and 3D effects

## Multi-Sheet Organization

### Recommended Sheet Structure for Proposals

| Tab Order | Sheet Name | Purpose | Color Tab |
|-----------|------------|---------|-----------|
| 1 | Cover | Title page, logo, date | Blue |
| 2 | Executive Summary | Key metrics, recommendation | Blue |
| 3 | Analysis | Detailed data and charts | Gray |
| 4 | Pricing | Cost breakdown, options | Green |
| 5 | Timeline | Implementation schedule | Gray |
| 6 | Appendix | Supporting data, methodology | No color |
| Last | Data (hidden) | Raw data, calculations | Red |

### Sheet Navigation Best Practices

1. **Use descriptive tab names** - Max 15 characters
2. **Color-code tab groups** - Use consistent colors for related sheets
3. **Hide technical sheets** - Right-click > Hide for raw data sheets
4. **Create a table of contents** - On Cover or Executive Summary sheet with hyperlinks
5. **Add navigation buttons** - Use shapes with hyperlinks between sheets

### Creating Hyperlink Navigation

```
=HYPERLINK("#'Executive Summary'!A1", "Go to Executive Summary")
```

Or use a shape/button:
1. Insert > Shapes > Rectangle
2. Right-click > Link > Place in This Document
3. Select destination sheet and cell

## Headers, Footers, and Branding

### Adding Professional Headers

**Left Section**: Company logo
**Center Section**: Document title
**Right Section**: Confidentiality notice or version

### How to Add a Logo to Header

1. Go to **Insert** > **Header & Footer**
2. Click in the Left Header section
3. Go to **Design** tab > **Picture**
4. Select your logo file
5. Resize using `&[Picture]` formatting codes

### Recommended Footer Elements

**Left**: Sheet name (`&[Tab]`)
**Center**: "Confidential - [Company Name]"
**Right**: Page number (`Page &[Page] of &[Pages]`)

### Dynamic Header/Footer Codes

| Code | Result |
|------|--------|
| `&[Page]` | Current page number |
| `&[Pages]` | Total page count |
| `&[Date]` | Current date |
| `&[Time]` | Current time |
| `&[Tab]` | Sheet name |
| `&[File]` | File name |
| `&[Path]&[File]` | Full file path |

### Applying Headers/Footers to Multiple Sheets

1. Hold **Ctrl** and click multiple sheet tabs (or right-click > Select All Sheets)
2. Go to **Page Layout** > **Page Setup** dialog
3. Click **Header/Footer** tab
4. Configure header/footer (applies to all selected sheets)

## Print-Ready Settings

### Recommended Margin Settings

| Margin | Setting | Purpose |
|--------|---------|---------|
| Top | 0.75" | Standard professional |
| Bottom | 0.75" | Standard professional |
| Left | 0.75" | Standard (or 1" for binding) |
| Right | 0.5" | Slightly less for balance |
| Header | 0.3" | Space for header content |
| Footer | 0.3" | Space for page numbers |

### Page Setup Checklist

- [ ] **Orientation**: Landscape for wide tables, Portrait for text-heavy
- [ ] **Scaling**: Fit to 1 page wide (let height auto-adjust)
- [ ] **Center on page**: Check both Horizontally and Vertically
- [ ] **Print titles**: Repeat header row on every page
- [ ] **Gridlines**: Uncheck "Print" option
- [ ] **Row/column headings**: Uncheck (don't print A, B, C... or 1, 2, 3...)

### How to Set Print Titles (Repeat Headers)

1. Go to **Page Layout** > **Print Titles**
2. In "Rows to repeat at top," select your header row (e.g., `$2:$2`)
3. Click OK

### Print Preview Best Practices

1. Always use **File > Print** to preview before finalizing
2. Enable **Show Margins** checkbox in Print Preview
3. Drag margin handles to fine-tune
4. Check for orphan data (single rows on new pages)
5. Insert manual page breaks where logical

## Quick Reference Cheat Sheet

### The 10 Commandments of Excel Proposals

1. **Leave Row 1 and Column A blank** - Data starts at B2
2. **Set zoom to 140%** - Better readability
3. **Use TableStyleMedium2** - Professional, not flashy
4. **Limit to 2-3 colors** - Primary, neutral, accent
5. **Remove all gridlines** - From both worksheets and charts
6. **Format numbers with commas** - `#,##0` for thousands
7. **Bold headers on dark background** - White text on `#2F5496`
8. **Right-align numbers, left-align text** - Never center data
9. **Add logo to header** - Consistent branding on every page
10. **Freeze panes** - Keep headers visible while scrolling
