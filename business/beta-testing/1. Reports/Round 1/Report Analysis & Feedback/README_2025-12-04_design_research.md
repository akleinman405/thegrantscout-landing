## SPECIFIC VALUES PROVIDED

### Colors (Hex Codes)
- Primary Navy: #1e3a5f
- Accent Gold: #d4a853
- Supporting neutrals: #ffffff, #fafbfc, #e8ebed, #b4b8c1
- Semantic: #36b37e (success), #ff8c42 (warning), #ff5630 (error), #0052cc (info)

### Typography
- Primary font: Inter
- Secondary font: Plus Jakarta Sans
- Code font: SF Mono
- Sizes: 40px, 32px, 28px, 24px, 20px, 16px, 14px, 12px, 11px
- Weights: 400 (regular), 500 (medium), 600 (semibold), 700 (bold)
- Line heights: 1.3 (headings), 1.5 (body), 1.4 (dense)

### Spacing (8px Grid)
- xs: 4px, sm: 8px, md: 16px, lg: 24px, xl: 32px, 2xl: 40px

### Measurements
- Body text: 14px minimum, 16px preferred
- Table padding: 12px vertical × 16px horizontal
- Callout border: 4-6px (left side)
- Border radius: 4px (sharp), 8px (medium), 12px (rounded)
- Box shadow: 0 2px 4px rgba(30,58,95,0.1) (subtle)

### Accessibility
- WCAG AA minimum: 4.5:1 contrast ratio
- Navy on white: 8.5:1 (AAA compliant)
- Gold on navy: 5.2:1 (AA compliant)
- Font size minimum: 12px (14px preferred)

## FREQUENTLY ASKED QUESTIONS

**Q: Should we use the navy + gold palette for everything?**
A: Yes. It's WCAG AA compliant, professional, and unique. Add neutrals and semantic colors as needed.

**Q: Do we have to use Inter and Plus Jakarta Sans?**
A: Preferred, but alternatives exist. The important thing is: 1 professional primary font, 1 secondary, consistent weights (400, 500, 600, 700).

**Q: What's the minimum font size?**
A: 12px for captions, but 14px body text is strongly recommended for readability.

**Q: How do we make tables mobile-friendly?**
A: Use responsive design - stack as cards on mobile (<768px). See THEGRANTSCOUT_IMPLEMENTATION_GUIDE.md Phase 6.

**Q: Is this design system locked in forever?**
A: No. Once built, you can iterate. But core colors, fonts, and spacing should remain consistent for 1-2 years minimum.

**Q: What about print reports?**
A: These recommendations work for both digital and print. PDF output will look identical to web version.
