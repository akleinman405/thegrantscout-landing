EXECUTIVE SUMMARY
─────────────────────────────────────────────────────────────────────────────

Modern professional business reports (2024-2025) are characterized by:
  • Bold minimalism (less is more, data is the hero)
  • Strategic typography (Inter + professional sans serif)
  • Navy + Gold color palettes (classic, professional, accessible)
  • Minimal table design (no gridlines, clear headers)
  • Generous whitespace (breathing room guides the eye)
  • Mobile-first responsive design (57% of traffic is mobile)
  • WCAG accessibility compliance (AA minimum standard)

TYPOGRAPHY SYSTEM
─────────────────────────────────────────────────────────────────────────────

FONTS:
  Headings:       Inter (geometric, professional, modern)
  Body:           Plus Jakarta Sans (readable, approachable)
  Code/Data:      SF Mono or Monaco (monospace for numbers)

SIZES & WEIGHTS:
  40px  weight 700  ← Page Title
  32px  weight 600  ← Section Header
  28px  weight 600  ← Major Heading
  24px  weight 600  ← Subheading
  20px  weight 500  ← Subheading (smaller)
  16px  weight 400  ← Body Text (large)
  14px  weight 400  ← Body Text (standard) ← DEFAULT
  12px  weight 400  ← Captions
  11px  weight 500  ← Fine Print

Line Heights:
  Headings:    1.3 × font size
  Body Text:   1.5 × font size (14px = 21px line height)

VISUAL HIERARCHY - THE 5 TECHNIQUES
─────────────────────────────────────────────────────────────────────────────

1. SIZE & SCALE
   Biggest element gets attention first
   40px > 28px > 16px (clear size differences matter)

2. COLOR & CONTRAST
   Bright gold on navy = spotlight effect
   Gold (#d4a853) on Navy (#1e3a5f) = 5.2:1 contrast (strong emphasis)

3. TYPOGRAPHY WEIGHT
   700 > 600 > 500 > 400 (bold stands out without size change)

4. WHITESPACE (Negative Space)
   Empty space frames importance
   40px margin around key metrics = "this matters"

5. ALIGNMENT & STRUCTURE
   Left-aligned body text (readable)
   Right-aligned numbers (scannable)
   Centered headings (emphasis)

CRITICAL RULE: Don't emphasize everything.
               Pick 1-2 focal points per page.
               When everything stands out, nothing does.

SPACING GRID (8px Base Unit)
─────────────────────────────────────────────────────────────────────────────

xs:   4px      (internal component spacing)
sm:   8px      (between list items)
md:   16px     (standard padding)
lg:   24px     (section padding)
xl:   32px     (major section separation)
2xl:  40px     (page breaks)

Examples:
  Card padding:       md (16px)
  Card margin:        lg (24px)
  Section margin:     xl-2xl (32-40px)
  Paragraph margin:   md (16px)
  Gap between items:  sm-md (8-16px)

RESPONSIVE DESIGN BREAKPOINTS
─────────────────────────────────────────────────────────────────────────────

Mobile (small):     < 480px
Mobile (large):     480px - 768px
Tablet:             768px - 1024px
Desktop:            1024px+

Strategy: Mobile-first (design for mobile, scale up for desktop)

Tables on Mobile:
  Desktop: Traditional columns
  Tablet:  Allow horizontal scroll
  Mobile:  Stack as cards (key-value pairs)

ACCESSIBILITY COMPLIANCE (WCAG AA)
─────────────────────────────────────────────────────────────────────────────

Contrast Ratios Required:
  Normal text:     4.5:1 minimum
  Large text:      3:1 minimum (18pt+ or 14pt bold)
  UI Components:   3:1 minimum

Your Palette Compliance:
  Navy on White:   8.5:1  ✓✓✓ Exceeds AAA
  Gold on Navy:    5.2:1  ✓✓  AA compliant
  Gold on White:   3.8:1  ✓   AA for large text only

Best Practices:
  • Don't use color as only indicator (add text labels)
  • Test with WebAIM Contrast Checker
  • Minimum 14px body font size
  • Test with colorblindness simulator
  • Icons paired with text (never icon-only)

COMMON MISTAKES TO AVOID
─────────────────────────────────────────────────────────────────────────────

❌ Too many fonts             (max 2: Inter + Plus Jakarta)
❌ Too many sizes             (max 4-5 different sizes)
❌ Everything bold            (defeats emphasis)
❌ Cramped spacing            (whitespace is good)
❌ Heavy drop shadows         (use subtle 0 2px 4px only)
❌ Light gray on white        (low contrast, unreadable)
❌ Icons without labels       (users won't understand)
❌ Tables with heavy gridlines (creates visual noise)
❌ Multiple focal points      (pick 1-2 per page)
