## Common Weaknesses vs. Framework

### Critical Missing Element: Funder Snapshots

**Framework Requirement:** Every opportunity must include 8-metric Funder Snapshot table:
1. Annual Giving ($X.XM across XX grants)
2. Typical Grant ($XX,000 median, range: $X,000 - $XXX,000)
3. Geographic Focus (XX% in-state)
4. Repeat Funding Rate (XX% funded 2+ times)
5. Giving Style (XX% general / XX% program-specific)
6. Recipient Profile (Typically $X-XM budget, sector focus)
7. Funding Trend (Growing/Stable/Declining with % change)
8. Comparable Grant (Org Name received $XX,000 for purpose, Year)

**Reality:** 0 of 5 reports include complete Funder Snapshots. All positioning advice is therefore generic, not data-backed.

## Recommendations for All Future Reports

### Immediate Fixes for Template

1. **Remove "About [Org]" section** - clients know their own organization
2. **Add Funder Snapshot template** with all 8 required metrics
3. **Convert timeline to week-by-week** (8 weeks, not 60 days)
4. **Require database citations** for all Funder Snapshot claims
5. **Add positioning data references** - every recommendation must cite a specific metric

### Quality Control Checklist

Before any report is delivered:
- [ ] All URLs manually tested (prevent SNS errors)
- [ ] All deadlines verified as current (prevent expired deadline issues)
- [ ] Grant program names verified against official sources
- [ ] Funder Snapshots populated with database queries
- [ ] Positioning strategies reference specific data points
- [ ] No "About [Org]" section included
- [ ] Timeline uses week-by-week format
