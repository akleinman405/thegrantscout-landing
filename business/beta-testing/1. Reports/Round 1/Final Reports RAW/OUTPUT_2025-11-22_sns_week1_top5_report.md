### Opportunity #1: California Senior Nutrition and Services Grant Program (CSNSGP)

**DEADLINE: December 12, 2025, 11:59 PM - URGENT**

#### Why This Opportunity

The CSNSGP offers substantial funding ($250,000-$500,000) with no match requirement from a $76 million statewide pool. While the fit score is conditional (50/100) pending eligibility confirmation, the funding amount and December 12 deadline make immediate assessment critical. SNS must determine if it serves populations at high risk for hate crimes to qualify.

#### Funder Overview

| Element | Details |
|---------|---------|
| Funder Name | California Governor's Office of Emergency Services (Cal OES) |
| Program | California State Nonprofit Security Grant Program (CSNSGP) |
| Funding Amount | $250,000 - $500,000 per award |
| Total Available | $76 million (2025-26 cycle) |
| Number of Awards | 1-300 statewide |
| Match Required | None |
| Deadline | December 12, 2025, 11:59 PM |
| Submission Method | Electronic to CSNSGP@caloes.ca.gov |

#### Application Process

**Step 1: Eligibility Assessment (By December 1)**
- [ ] Determine if SNS serves populations at high risk for hate crimes
- [ ] Review client demographics for qualifying populations:
  - LGBTQ+ older adults
  - Immigrant/refugee seniors
  - Seniors with disabilities targeted by bias
  - Religious minority seniors
  - Racial/ethnic minority seniors
- [ ] Document services to at-risk populations
- [ ] If uncertain, contact CSNSGP Unit at 916-845-8410

**Step 2: Application Preparation (December 1-10)**
- [ ] Download Application Form (Version 10/25)
- [ ] Complete Vulnerability Assessment (VA) Worksheet (Version 10/25)
- [ ] Gather supporting documents demonstrating high-risk status
- [ ] Prepare security assessment documentation
- [ ] Develop budget for proposed security enhancements

**Step 3: Application Submission (By December 12)**
- [ ] Compile complete application package
- [ ] Submit electronically to CSNSGP@caloes.ca.gov
- [ ] Confirm receipt of submission
- [ ] Retain copies of all submitted materials

#### Required Documents Checklist

- [ ] Application Form (Version 10/25)
- [ ] Vulnerability Assessment (VA) Worksheet (Version 10/25)
- [ ] IRS 501(c)(3) determination letter
- [ ] Supporting documents demonstrating high-risk status
- [ ] Security assessment or needs documentation
- [ ] Budget for proposed security enhancements
- [ ] Organizational profile

#### Key Contacts

| Contact | Details |
|---------|---------|
| CSNSGP Unit | 916-845-8410 |
| Email | CSNSGP@caloes.ca.gov |
| Website | https://www.caloes.ca.gov/ |

#### Positioning Strategy

If SNS determines eligibility, emphasize:
- Documented services to at-risk senior populations
- Physical facility security needs
- Community gathering point function
- Staff and client safety concerns

#### Funder Intelligence

- Applications scored on six criteria totaling 48 points
- Organizations must score 40+ points to be considered
- Scoring based on: threat assessment, vulnerability assessment, consequence assessment, organizational capacity, project feasibility, budget justification
- Grant operates on reimbursement basis (spend first, then request reimbursement)

#### Application Tips

- Focus documentation on populations at high risk for hate crimes
- Be specific about security vulnerabilities and proposed solutions
- Include detailed cost estimates for security enhancements
- If eligibility is uncertain, contact CSNSGP Unit before deadline

#### Estimated Time to Complete

- Eligibility assessment: 2-4 hours
- Application preparation: 15-20 hours
- Document gathering: 5-10 hours
- Total: 25-35 hours
