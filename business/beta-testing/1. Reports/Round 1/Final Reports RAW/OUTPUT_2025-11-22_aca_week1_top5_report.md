### Overview

| Field | Detail |

### Why This Opportunity

Walmart Spark Good is an accessible entry point for grant funding with relatively simple application requirements. Key advantages:

- **Multiple Applications Allowed:** Up to 25 store requests per year
- **Local Decision Making:** Store managers decide awards (relationship building opportunity)
- **Equipment Focus:** Athletic equipment is eligible expense
- **Quick Turnaround:** Quarterly review cycles
- **Low Competition:** Many eligible nonprofits don't know about this program

### Funder Overview

| Category | Details |
|----------|---------|
| **Funder** | Walmart Foundation |
| **Program** | Spark Good Local Community Grants |
| **Focus Areas** | Youth development, community benefit, local impact |
| **Award Cycle** | Quarterly (Current: Nov 3 - Dec 31, 2025) |
| **Decision Makers** | Local store managers |

### Application Process

**Step 1: Create Walmart.com Account (Day 1)**
- Go to walmart.com/nonprofits
- Click "Get Started" button
- Create account using organizational email (not personal)
- Use official @arborbrook email address

**Step 2: Access Spark Good Platform (Day 1)**
- Click "Access Spark Good"
- Select "Add organization" to claim your profile
- Enter EIN: 20-2707577

**Step 3: Complete DEED Verification (Days 1-14)**
- DEED is Walmart's third-party verification partner
- Submit required documentation for verification
- Allow 1-2 weeks for verification to complete
- Technical support: nonprofits@joindeed.com

**Step 4: Set Up Payment Method (After Verification)**
- Navigate to Settings > Payments
- Set up PayPal or Network for Good account
- Required for fund receipt

**Step 5: Submit Grant Applications (After Verification)**
- Apply to local Walmart stores in Matthews area
- Each store can provide separate grant ($250-$5,000)
- Target multiple stores: Matthews, Indian Trail, Mint Hill, Monroe

### Required Documents Checklist

- [ ] EIN (20-2707577)
- [ ] IRS 501(c)(3) determination letter
- [ ] Organizational email address
- [ ] PayPal account for fund receipt
- [ ] Brief program description (equipment needs)

Lead With:
