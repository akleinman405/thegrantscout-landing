## About This Tracker

This application tracker helps Ka Ulukoa manage multiple facility funding applications simultaneously by organizing opportunities into four stages. The tracker was created from the 10 validated opportunities in your November 2025 Grant Opportunity Report, with emphasis on the January 2026 priority deadlines (Atherton Family Foundation Jan 5, Bank of Hawaii Foundation Jan 16, USATF Foundation Jan 31). Review this tracker weekly and update status as applications progress. Given your facility acquisition goals, special attention is given to layered funding strategy.

## Key Documents Master Checklist

### Core Organization Documents
- [ ] 501(c)(3) determination letter (PDF ready)
- [ ] Financial statements - Year 1 (audited if available)
- [ ] Financial statements - Year 2 (audited if available)
- [ ] Board of directors list with affiliations
- [ ] Organizational budget (current fiscal year)
- [ ] Annual report or impact summary

### Facility Project Documents
- [ ] Project budget (detailed line items)
- [ ] Architectural renderings or conceptual plans
- [ ] Cost estimates from licensed contractors
- [ ] Project timeline with milestones
- [ ] Sustainability/maintenance plan
- [ ] Site control documentation (lease, deed, etc.)

### Facility Acquisition Documentation
- [ ] Property appraisal (if applicable)
- [ ] Purchase agreement or letter of intent
- [ ] Site survey and boundary documentation
- [ ] Zoning compliance verification
- [ ] Title search/insurance documentation

### Community Support Documentation
- [ ] Letters of support from families served
- [ ] Letters from partner organizations
- [ ] Letters from community leaders
- [ ] Endorsements from schools/education partners
- [ ] Media coverage or recognition

### Population & Impact Documentation
- [ ] Demographics of youth served
- [ ] Native Hawaiian participation data (CRITICAL)
- [ ] Low/moderate income verification
- [ ] Geographic service area map
- [ ] Current program participation numbers
- [ ] Projected facility usage statistics

### Match & Leverage Documentation
- [ ] Proof of 60% funding secured (for Atherton if >$500K)
- [ ] Committed funds letters from other funders
- [ ] Pending grant applications list
- [ ] Volunteer hour tracking system
- [ ] In-kind donation valuations
- [ ] Fundraising event history

### Native Hawaiian Program Component Documentation
- [ ] Cultural practices integration description
- [ ] Hawaiian language usage documentation
- [ ] Partnership with Native Hawaiian organizations
- [ ] Cultural advisor involvement
- [ ] Native Hawaiian board/staff representation
