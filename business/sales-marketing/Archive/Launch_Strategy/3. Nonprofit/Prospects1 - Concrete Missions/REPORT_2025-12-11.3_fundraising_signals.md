## Executive Summary

We analyzed F990 data to identify signals that predict active fundraising activity among our 642 prime prospects. While the IRS 990 forms lack explicit fundraising fields (no Schedule G data, no special events revenue, no fundraising expenses breakdown), we identified 7 proxy signals that correlate with active fundraising behavior.

**Key Findings:**
- Created a 10-point `fundraising_likelihood_score` using 7 signals
- 119 prospects (18.5%) score 7+ (high likelihood)
- 216 prospects (33.6%) score 6+ (medium-high likelihood)
- Validation hit rate: **10 of 15 (67%)** of top-scoring orgs had active fundraising on their websites

**Recommendation:** Prioritize prospects with score >= 6 for Phase 4 web scraping (216 organizations).

## 3. Scoring Model

### Formula: `fundraising_likelihood_score` (0-10 points)

| Signal | Points | Criteria |
|--------|--------|----------|
| December Fiscal YE | 2 | `fiscal_year_end_month = 12` |
| High Grant Dependency | 2 | `grant_dependency_pct >= 90%` |
| High Volunteer Engagement | 2 | `volunteer_count / employee_count >= 10` |
| Diverse Funder Base | 1 | `num_unique_funders >= 5` |
| Active Grant Receiver | 1 | `num_foundation_grants >= 3` |
| Low Program Revenue | 1 | `program_service_revenue / total_revenue < 25%` |
| High Fundraising Sector | 1 | `sector IN ('A','D','E','G','K','L','O','P')` |
| **Total Possible** | **10** | |

### SQL Implementation
```sql
ALTER TABLE f990_2025.prospects
ADD COLUMN IF NOT EXISTS fundraising_likelihood_score SMALLINT DEFAULT 0;

UPDATE f990_2025.prospects p
SET fundraising_likelihood_score = (
    -- December FYE (2 pts)
    CASE WHEN EXTRACT(MONTH FROM nr.tax_period_end) = 12 THEN 2 ELSE 0 END
    -- High grant dependency (2 pts)
    + CASE WHEN p.grant_dependency_pct >= 90 THEN 2 ELSE 0 END
    -- High volunteer engagement (2 pts)
    + CASE WHEN p.volunteer_count > 0 AND p.employee_count > 0
           AND (1.0 * p.volunteer_count / p.employee_count) >= 10 THEN 2 ELSE 0 END
    -- Diverse funders (1 pt)
    + CASE WHEN p.num_unique_funders >= 5 THEN 1 ELSE 0 END
    -- Active grants (1 pt)
    + CASE WHEN p.num_foundation_grants >= 3 THEN 1 ELSE 0 END
    -- Low program revenue (1 pt)
    + CASE WHEN p.total_revenue > 0
           AND (p.program_service_revenue / p.total_revenue) < 0.25 THEN 1 ELSE 0 END
    -- High fundraising sector (1 pt)
    + CASE WHEN p.sector IN ('A','D','E','G','K','L','O','P') THEN 1 ELSE 0 END
)
FROM f990_2025.nonprofit_returns nr
WHERE p.ein = nr.ein AND p.tax_year = nr.tax_year;
```
