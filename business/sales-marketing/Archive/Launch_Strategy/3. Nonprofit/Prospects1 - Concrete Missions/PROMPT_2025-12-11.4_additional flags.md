# Claude Code CLI Prompt
## Add Remaining Prospect Flags

---

## Context & Goal

**Situation:** We have a prospects table with 74,144 nonprofits and several flags already built (ICP score, reachability, concrete mission, capacity grant, fundraising likelihood score). We need to add the remaining flags so we can mix and match filters for outreach targeting.

**Goal:** Add 5 new flags to the prospects table:
1. `likely_active_fundraiser` — 0/1 flag from existing score
2. `fiscal_year_end_soon` — 0/1 flag for orgs with fiscal year ending in 14-60 days
3. `has_biz_dev_staff` — yes/no/uncertain based on officer titles
4. `new_to_foundations` — 0/1 flag for orgs with zero foundation grants
5. `shares_funder_with_beta` — 0/1 flag (convert existing text field)

**Database:** PostgreSQL, schema `f990_2025`, Windows machine, connect via Python/psycopg2 (no psql client).

## Flag 1: likely_active_fundraiser

Convert existing `fundraising_likelihood_score` to 0/1 flag using threshold of 6.

```sql
ALTER TABLE f990_2025.prospects 
ADD COLUMN IF NOT EXISTS likely_active_fundraiser SMALLINT DEFAULT 0;

UPDATE f990_2025.prospects
SET likely_active_fundraiser = CASE 
    WHEN fundraising_likelihood_score >= 6 THEN 1 
    ELSE 0 
END;
```

## Flag 2: fiscal_year_end_soon

Flag orgs with fiscal year ending in the next 14-60 days. Today is 2025-12-11.

**Step 1:** Check fiscal year end data

```sql
-- See what fiscal year end data looks like
SELECT 
    EXTRACT(MONTH FROM nr.tax_period_end) as fy_month,
    COUNT(*) as cnt
FROM f990_2025.prospects p
JOIN f990_2025.nonprofit_returns nr ON p.ein = nr.ein AND p.tax_year = nr.tax_year
GROUP BY 1
ORDER BY 1;
```

**Step 2:** Add column and flag

```sql
ALTER TABLE f990_2025.prospects 
ADD COLUMN IF NOT EXISTS fiscal_year_end_soon SMALLINT DEFAULT 0;

-- Flag if fiscal year end month is within 14-60 days of today (Dec 11)
-- Dec 11 + 14 days = Dec 25
-- Dec 11 + 60 days = Feb 9
-- So flag December (12), January (1), February (2) fiscal year ends

UPDATE f990_2025.prospects p
SET fiscal_year_end_soon = 1
FROM f990_2025.nonprofit_returns nr
WHERE p.ein = nr.ein 
  AND p.tax_year = nr.tax_year
  AND EXTRACT(MONTH FROM nr.tax_period_end) IN (12, 1, 2);
```

**Note:** This is a simplified approach using month only. A more precise approach would calculate exact days, but month is sufficient for targeting purposes.

## Flag 3: has_biz_dev_staff

Check officer/compensation data for development-related titles.

**Step 1:** Find the officer data table

```sql
-- Find tables with officer/compensation data
SELECT table_name 
FROM information_schema.tables 
WHERE table_schema = 'f990_2025' 
  AND (table_name ILIKE '%officer%' 
       OR table_name ILIKE '%comp%' 
       OR table_name ILIKE '%person%'
       OR table_name ILIKE '%part_vii%');

-- Check columns
SELECT column_name 
FROM information_schema.columns 
WHERE table_schema = 'f990_2025' 
  AND table_name = '[officer_table_name]';
```

**Step 2:** Check for development titles

Keywords indicating business development/fundraising staff:
- development
- fundrais
- advancement
- grant writer
- donor
- philanthrop
- major gift
- annual fund
- capital campaign

```sql
-- Sample officer titles to understand data
SELECT DISTINCT title, COUNT(*) as cnt
FROM f990_2025.[officer_table]
WHERE title IS NOT NULL
GROUP BY title
ORDER BY cnt DESC
LIMIT 50;

-- Check for development-related titles
SELECT ein, name, title
FROM f990_2025.[officer_table]
WHERE LOWER(title) LIKE '%develop%'
   OR LOWER(title) LIKE '%fundrais%'
   OR LOWER(title) LIKE '%advancement%'
   OR LOWER(title) LIKE '%grant%'
   OR LOWER(title) LIKE '%donor%'
   OR LOWER(title) LIKE '%philanthrop%'
   OR LOWER(title) LIKE '%major gift%'
   OR LOWER(title) LIKE '%annual fund%'
LIMIT 20;
```

**Step 3:** Add column and flag

```sql
ALTER TABLE f990_2025.prospects 
ADD COLUMN IF NOT EXISTS has_biz_dev_staff VARCHAR(10) DEFAULT 'uncertain';

-- Flag as 'yes' if development title found
UPDATE f990_2025.prospects p
SET has_biz_dev_staff = 'yes'
FROM f990_2025.[officer_table] o
WHERE p.ein = o.ein
  AND (
    LOWER(o.title) LIKE '%develop%'
    OR LOWER(o.title) LIKE '%fundrais%'
    OR LOWER(o.title) LIKE '%advancement%'
    OR LOWER(o.title) LIKE '%grant writer%'
    OR LOWER(o.title) LIKE '%donor%'
    OR LOWER(o.title) LIKE '%philanthrop%'
    OR LOWER(o.title) LIKE '%major gift%'
  );

-- Flag as 'no' if we have officer data but no dev title found
UPDATE f990_2025.prospects p
SET has_biz_dev_staff = 'no'
WHERE has_biz_dev_staff = 'uncertain'
  AND EXISTS (
    SELECT 1 FROM f990_2025.[officer_table] o 
    WHERE o.ein = p.ein
  );

-- Remaining 'uncertain' = no officer data available
```

## Flag 4: new_to_foundations

Flag orgs with zero foundation grants in our database.

```sql
ALTER TABLE f990_2025.prospects 
ADD COLUMN IF NOT EXISTS new_to_foundations SMALLINT DEFAULT 0;

UPDATE f990_2025.prospects

    WHEN num_foundation_grants = 0 OR num_foundation_grants IS NULL THEN 1 
    ELSE 0 
END;
```

## Flag 5: shares_funder_with_beta

Convert existing `beta_client_connection` text field to 0/1 flag.

**Step 1:** Check current field

```sql
-- See what's in the beta_client_connection field
SELECT beta_client_connection, COUNT(*) as cnt
FROM f990_2025.prospects
WHERE beta_client_connection IS NOT NULL
GROUP BY beta_client_connection
LIMIT 20;
```

**Step 2:** Add flag

```sql
ALTER TABLE f990_2025.prospects
```

## Output Summary

After adding all flags, run summary:

```sql
SELECT 
    COUNT(*) as total_prospects,
    SUM(likely_active_fundraiser) as likely_fundraiser,
    SUM(fiscal_year_end_soon) as fy_end_soon,
    SUM(CASE WHEN has_biz_dev_staff = 'yes' THEN 1 ELSE 0 END) as has_biz_dev_yes,

    SUM(shares_funder_with_beta) as shares_funder_beta
FROM f990_2025.prospects;

-- Cross-tab with concrete mission
SELECT 
    has_concrete_mission,
    SUM(likely_active_fundraiser) as likely_fundraiser,
    SUM(fiscal_year_end_soon) as fy_end_soon,
    SUM(new_to_foundations) as new_to_foundations,
    COUNT(*) as total
FROM f990_2025.prospects
GROUP BY has_concrete_mission;

-- Prime prospects with all flags
SELECT COUNT(*) as prime_count
FROM f990_2025.prospects
WHERE has_concrete_mission = 1
  AND likely_active_fundraiser = 1
  AND fiscal_year_end_soon = 1
  AND has_biz_dev_staff = 'no'
  AND employee_count BETWEEN 0 AND 3;
```
