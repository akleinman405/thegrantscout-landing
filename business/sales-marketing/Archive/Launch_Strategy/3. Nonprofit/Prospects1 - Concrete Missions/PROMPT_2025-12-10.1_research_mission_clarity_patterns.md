# Claude Code CLI Prompt
## Identify "Simple Mission" Nonprofit Patterns

**Goal:** Find database signals that predict which nonprofits have clear, concrete, and simple missions ("We help X by doing Y") vs complex multi-program orgs. (example: we help vets overcome PTSD through sailing)

---

## Phase 1: Sample Diverse Nonprofits (n=50)

Pull 50 prospects stratified across:
- Different NTEE sectors (E, P, B, N, A, etc.)
- Different revenue bands
- Different employee counts
- Mix of single vs multiple NTEE codes (if detectable)

```sql
SELECT ein, org_name, state, sector, ntee_code, total_revenue, employee_count
FROM f990_2025.prospects
WHERE icp_score >= 8
ORDER BY RANDOM()
LIMIT 50;
```

---

## Phase 2: Research Each Org

For each org, visit website and record:

| Field | How to Code |
|-------|-------------|
| mission_clarity | 1 = Clear one-liner / 2 = Moderate / 3 = Complex |
| mission_structure | 1 = "We help X by doing Y" / 0 = Other |
| program_count | 1 / 2-3 / 4+ |
| purpose_word_count | Count words in mission statement |
| has_active_fundraiser | 1 = Yes / 0 = No |
| org_name_descriptive | 1 = Name describes what they do / 0 = Abstract name |

Also extract:
- mission_text (verbatim)
- one_liner (your summary in <10 words)

---

## Phase 3: Analyze Patterns

Look for correlations between mission_clarity (1 = clear) and:

- NTEE sector
- Single vs multiple NTEE codes
- Employee count
- Revenue
- Org name characteristics
- Purpose statement length (from F990 if available)
- Keywords in org name (e.g., "veterans", "youth", "shelter")

```python
# Cross-tab examples
pd.crosstab(df['sector'], df['mission_clarity'])
pd.crosstab(df['employee_count_band'], df['mission_clarity'])

# Correlation with numeric fields
df[['mission_clarity', 'employee_count', 'total_revenue', 'purpose_word_count']].corr()
```

**Hypotheses to test:**
- Smaller orgs (< 20 employees) have clearer missions?
- Certain sectors (N=recreation, P=human services) more concrete?
- Descriptive org names correlate with clear missions?
- Shorter F990 purpose statements = clearer missions?

---

## Phase 4: Validate Patterns

Once patterns emerge, pull 30 NEW prospects matching the pattern and verify:

Example: If "sector N + <15 employees + descriptive name" predicts clarity:

```sql
SELECT ein, org_name, sector, employee_count
FROM f990_2025.prospects
WHERE sector = 'N'
  AND employee_count < 15
  AND icp_score >= 8
  AND ein NOT IN (... first 50 ...)
ORDER BY RANDOM()
LIMIT 30;
```

Research these 30 and check: What % have clear missions?

If >70% = pattern confirmed. If <50% = pattern doesn't hold.
