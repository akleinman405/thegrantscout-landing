## Step 5: Create Fundraising Likelihood Score

Based on findings, create a scoring model:

```sql
-- Example scoring (adjust based on Step 4 findings)
ALTER TABLE f990_2025.prospects ADD COLUMN IF NOT EXISTS fundraising_likelihood_score SMALLINT DEFAULT 0;

UPDATE f990_2025.prospects p
SET fundraising_likelihood_score = (
    -- Has special events revenue
    CASE WHEN nr.special_events_revenue > 0 THEN 2 ELSE 0 END
    -- Has fundraising expenses > 5%

    -- Q4 fiscal year end (likely year-end campaign)
```
