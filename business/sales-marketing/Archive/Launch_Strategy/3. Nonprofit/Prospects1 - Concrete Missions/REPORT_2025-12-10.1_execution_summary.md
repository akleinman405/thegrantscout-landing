# Execution Report: Mission Clarity Research

**Date:** 2025-12-10
**Prompt:** PROMPT_2025-12-10_research_mission_clarity_patterns.md

---

## What Was Done

### Phase 1: Database Sampling
- Queried `f990_2025.prospects` table for 50 diverse nonprofits
- Stratified by: sector, employee count band, revenue band
- Required: ICP score >= 7, website available, mission statement present

### Phase 2: Website Research (Parallelized)
- Launched 3 scout agents simultaneously
  - Agent 1: Orgs 1-17
  - Agent 2: Orgs 18-34
  - Agent 3: Orgs 35-50
- Each agent visited websites and coded:
  - mission_clarity (1=clear concrete, 2=moderate, 3=complex)
  - mission_structure (1="We do X for Y", 0=other)
  - program_count (1, 2-3, 4+)
  - org_name_descriptive (1=name describes purpose, 0=abstract)
  - beneficiary, action, one_liner

### Phase 3: Pattern Analysis
- Consolidated data from 3 agents
- Calculated correlation between clarity and:
  - Sector
  - Employee count
  - Name descriptiveness
  - Mission structure
  - Program count

### Phase 4: Validation
- Pulled 30 NEW prospects matching discovered pattern (beneficiary keywords)
- Assessed clarity based on 990 mission text
- Confirmed 90% success rate

### Phase 5: Deliverables
- Created final report with SQL filter
- Generated CSV data file with all 50 coded organizations

---

## What Was Learned

### Key Findings

1. **"We do X for Y" structure is 89% predictive** of clear missions (vs 0% for other structures)

2. **Beneficiary keywords in org name = 90% clear** - Keywords like "children", "youth", "shelter", "food bank", "rescue", "veterans" predict concrete missions

3. **Descriptive names = 80% clear** - Names that describe what org does (vs abstract names like "Horizons") strongly predict clarity

4. **Mid-sized orgs (16-50 employees) = 91% clear** - Sweet spot for mission clarity

5. **Program count doesn't matter much** - Orgs with 8 programs can still be clear if they serve specific population

### Surprising Findings

| Expected | Actual |
|----------|--------|
| Smaller orgs = clearer | Mid-sized (16-50) are clearest |
| Fewer programs = clearer | Program count is weak signal |
| Short missions = clearer | Structure matters more than length |
| Certain sectors = clearer | Sector is weak signal |

## How It Was Learned

### Methodology That Worked

1. **Parallel agent execution** - 3 scouts researching websites simultaneously cut research time from ~90 min to ~30 min

2. **Database pre-analysis** - Running pattern queries while agents worked (mission length by sector, keyword counts) maximized efficiency

3. **Quick validation** - Using 990 mission text for validation sample (vs full website research) was sufficient to confirm patterns

4. **Coded data approach** - Structured coding (1/2/3 scale) enabled quantitative correlation analysis

## Unexpected Choices

### 1. Shifted Focus Mid-Research

**Original prompt:** Find orgs with "simple" missions
**User clarification:** Priority is CLEAR/CONCRETE missions, not necessarily short

**Impact:** Re-coded data focusing on "We do X for Y people" pattern rather than word count

### 2. Used 990 Text for Validation Instead of Full Website Research

**Rationale:** The 990 mission statements were sufficient to assess "clear vs vague" - didn't need full website deep dive for validation sample

**Time saved:** ~45 min of additional agent runtime

### 3. Excluded "Foundation" Suffix from Recommended Filter

**Discovery:** Organizations ending in "Foundation" that support another institution (e.g., "Iowa Law School Foundation") have vague missions because they're fundraising vehicles, not direct-service providers

## Lessons Learned (Wish I'd Known Going In)

### 1. Name Analysis is Faster Than Website Research

**Lesson:** Could have started with name-based keyword analysis, then only researched websites for ambiguous cases. Would have cut research time by 50%.

**For next time:** Start with database-only analysis of org names, 990 mission text before launching website research agents.

### 2. "Foundation" is a Red Flag

**Lesson:** The word "Foundation" at the end of an org name often indicates a supporting organization with a vague "support X institution" mission - not a direct-service provider with concrete beneficiaries.

**For next time:** Filter out "Foundation$" suffix from simple-mission searches.

### 3. Concrete Nouns > Abstract Verbs

**Lesson:** Clear missions use concrete nouns (shelter, food, housing, education) + specific populations (homeless families, disabled veterans). Vague missions use abstract verbs (empower, transform, inspire) + generic populations (community, future generations).

**For next time:** Can detect this programmatically by looking for abstract verb patterns in mission text.

### 4. Geographic Specificity Signals Clarity

**Lesson:** Organizations that name their geography ("Licking County", "Central Coast", "Nisqually Watershed") tend to have clearer missions than those claiming national/global scope.

**For next time:** Consider geographic specificity as an additional filter signal.

### 5. Website Research Has Diminishing Returns

**Lesson:** Website research confirmed what 990 mission text already showed. The extra detail (program count, fundraiser visibility) didn't significantly change clarity assessments.

**For next time:** Use 990 mission text analysis as primary method; reserve website research for edge cases.
