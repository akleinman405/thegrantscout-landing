# Claude Code CLI Prompt
## Add Reachability Flags to Prospects Table

**Goal:** Add dummy variables (0/1) for reachability signals, plus a combined flag.

---

## Add Columns

```sql
ALTER TABLE f990_2025.prospects ADD COLUMN IF NOT EXISTS reach_employees_1_10 SMALLINT DEFAULT 0;
ALTER TABLE f990_2025.prospects ADD COLUMN IF NOT EXISTS reach_revenue_1m_2m SMALLINT DEFAULT 0;
ALTER TABLE f990_2025.prospects ADD COLUMN IF NOT EXISTS reach_form_990 SMALLINT DEFAULT 0;
ALTER TABLE f990_2025.prospects ADD COLUMN IF NOT EXISTS reach_sector_e_p SMALLINT DEFAULT 0;
ALTER TABLE f990_2025.prospects ADD COLUMN IF NOT EXISTS reach_grant_dep_75 SMALLINT DEFAULT 0;
ALTER TABLE f990_2025.prospects ADD COLUMN IF NOT EXISTS reach_more_employees_than_volunteers SMALLINT DEFAULT 0;
ALTER TABLE f990_2025.prospects ADD COLUMN IF NOT EXISTS reach_all_flags SMALLINT DEFAULT 0;
```

## Populate Each Flag

```sql
UPDATE f990_2025.prospects SET reach_employees_1_10 = CASE WHEN employee_count BETWEEN 1 AND 10 THEN 1 ELSE 0 END;
UPDATE f990_2025.prospects SET reach_revenue_1m_2m = CASE WHEN total_revenue BETWEEN 1000000 AND 2000000 THEN 1 ELSE 0 END;
UPDATE f990_2025.prospects SET reach_form_990 = CASE WHEN form_type = '990' THEN 1 ELSE 0 END;
UPDATE f990_2025.prospects SET reach_sector_e_p = CASE WHEN sector IN ('E', 'P') THEN 1 ELSE 0 END;
UPDATE f990_2025.prospects SET reach_grant_dep_75 = CASE WHEN grant_dependency_pct >= 75 THEN 1 ELSE 0 END;
UPDATE f990_2025.prospects SET reach_more_employees_than_volunteers = CASE WHEN employee_count > COALESCE(volunteer_count, 0) THEN 1 ELSE 0 END;
```

## Calculate Combined Flag

```sql
UPDATE f990_2025.prospects 
SET reach_all_flags = CASE 
    WHEN reach_employees_1_10 = 1 
     AND reach_revenue_1m_2m = 1 
     AND reach_form_990 = 1 
     AND reach_sector_e_p = 1 
     AND reach_grant_dep_75 = 1 
     AND reach_more_employees_than_volunteers = 1 
    THEN 1 
    ELSE 0 
END;
```

## Output Summary

```sql
SELECT 
    SUM(reach_employees_1_10) as employees_1_10,
    SUM(reach_revenue_1m_2m) as revenue_1m_2m,
    SUM(reach_form_990) as form_990,
    SUM(reach_sector_e_p) as sector_e_p,
    SUM(reach_grant_dep_75) as grant_dep_75,
    SUM(reach_more_employees_than_volunteers) as more_emp_than_vol,
    SUM(reach_all_flags) as all_flags
FROM f990_2025.prospects;

-- Top prospects with all flags
SELECT ein, org_name, state, sector, total_revenue, employee_count, icp_score
FROM f990_2025.prospects
WHERE reach_all_flags = 1
ORDER BY icp_score DESC
LIMIT 20;
```
