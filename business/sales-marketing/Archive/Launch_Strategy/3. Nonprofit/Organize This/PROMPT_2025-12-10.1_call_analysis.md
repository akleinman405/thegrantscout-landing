## Known Columns

```
Organization_Name, Phone, Officer_Name, Officer_Title, Address, City, State, ZIP, 
Website, Total_Revenue, Grants_Revenue, Grant_Dependency_Pct, Total_Expenses,

Send us an Email, Reached Decision Maker, Yes/No/Maybe/Uncertain, Contact Date, Notes
```

**Key Outcome Columns:**
- `Talked to Someone` — Connected with a human (any human)
- `Reached Decision Maker` — Got to the right person
- `Yes/No/Maybe/Uncertain` — Interest level outcome
- `VM/Message` — Left voicemail
- `Send us an Email` — Asked to email instead

**Key Predictor Columns (already in file):**
- `Total_Revenue` — Org size
- `Grants_Revenue` — Foundation funding received
- `Grant_Dependency_Pct` — How reliant on grants
- `NTEE_Codes` — Sector
- `State` — Geography
- `Max Weekly Hours` — Possibly org capacity indicator
- `Prospect_Score` — Existing scoring (compare to outcomes)

## Step 2: Create Binary Outcome Flags

```python
# Convert outcome columns to binary (handle various formats: Yes/No, Y/N, 1/0, TRUE/FALSE, etc.)
def to_binary(col):
    return col.astype(str).str.lower().str.strip().isin(['yes', 'y', '1', 'true', 'x']).astype(int)

calls['connected'] = to_binary(calls['Talked to Someone'])
calls['reached_dm'] = to_binary(calls['Reached Decision Maker'])
calls['left_vm'] = to_binary(calls['VM/Message'])
calls['email_requested'] = to_binary(calls['Send us an Email'])

# Interest level
calls['interested'] = calls['Yes/No/Maybe/Uncertain'].astype(str).str.lower().str.strip().isin(['yes', 'maybe']).astype(int)

print(f"\n--- Connection Rates ---")
print(f"Connected with anyone: {calls['connected'].mean():.1%}")
print(f"Reached decision maker: {calls['reached_dm'].mean():.1%}")
print(f"Left voicemail: {calls['left_vm'].mean():.1%}")
print(f"Interest (Yes/Maybe): {calls['interested'].mean():.1%}")
```

# Revenue bands
calls['revenue_band'] = pd.cut(

## Step 10: Build Ideal Customer Profile from Beta Clients

```python
print("\n=== IDEAL CUSTOMER PROFILE (from Beta Clients) ===")

beta_clients = calls[calls['is_beta_client']]

if len(beta_clients) >= 3:
    # Calculate ranges and common attributes
    icp = {
        'revenue_min': beta_clients['Total_Revenue'].min(),
        'revenue_max': beta_clients['Total_Revenue'].max(),
        'revenue_median': beta_clients['Total_Revenue'].median(),
        'grant_dep_min': beta_clients['Grant_Dependency_Pct'].min(),
        'grant_dep_max': beta_clients['Grant_Dependency_Pct'].max(),
        'sectors': beta_clients['sector'].unique().tolist(),
        'states': beta_clients['State'].unique().tolist(),
    }
    
    print(f"""
IDEAL CUSTOMER PROFILE (Based on {len(beta_clients)} Beta Clients):

Revenue:        ${icp['revenue_min']:,.0f} - ${icp['revenue_max']:,.0f}
                (Median: ${icp['revenue_median']:,.0f})

Grant Dependency: {icp['grant_dep_min']:.0f}% - {icp['grant_dep_max']:.0f}%

Sectors (NTEE):  {', '.join(icp['sectors'])}

States:          {', '.join(icp['states'])}

RECOMMENDATION: Use these criteria to filter the F990 database for 
high-probability prospects that match your successful beta clients.
""")

    # Score all prospects based on similarity to beta profile
    calls['icp_score'] = 0
    
    # Revenue in range: +2 points
    calls.loc[
        (calls['Total_Revenue'] >= icp['revenue_min']) & 
        (calls['Total_Revenue'] <= icp['revenue_max']), 
        'icp_score'
    ] += 2
    
    # Sector match: +2 points
    calls.loc[calls['sector'].isin(icp['sectors']), 'icp_score'] += 2
    
    # Grant dependency in range: +1 point
    calls.loc[
        (calls['Grant_Dependency_Pct'] >= icp['grant_dep_min']) & 
        (calls['Grant_Dependency_Pct'] <= icp['grant_dep_max']), 
        'icp_score'
    ] += 1
    
    print("\n--- ICP Score Distribution ---")
    print(calls.groupby('icp_score')['connected'].agg(['mean', 'count']))
    
    print("\n--- Top Non-Beta Prospects (Highest ICP Score) ---")
    top_prospects = calls[~calls['is_beta_client']].nlargest(10, 'icp_score')[
        ['Organization_Name', 'State', 'Total_Revenue', 'sector', 'icp_score', 'connected', 'interested']
    ]
    print(top_prospects.to_string(index=False))
```
