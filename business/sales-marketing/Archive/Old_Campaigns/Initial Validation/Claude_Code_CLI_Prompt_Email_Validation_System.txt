ANTI-SPAM RANDOMIZATION (CRITICAL!)
===============================================================================

Make sending patterns look human to avoid spam detection:

RANDOM DELAYS:
- Base delay between emails: random.uniform(5, 10) seconds
- Calculate target pace from remaining emails and time
- Add variation: ±15% from target pace
- Micro-jitter: ±2 seconds on every send
- Random longer pauses: Every 10-20 emails, wait 30-90 seconds 
  (simulate "coffee break")

HUMAN-LIKE PATTERNS:
- First 30 minutes: Slightly slower (simulate "warming up")
- Middle of day: Normal pace
- Last hour: Slightly slower (simulate "wrapping up")
- Never perfectly spaced intervals

PACING CALCULATION EXAMPLE:
```python
# If we have 300 emails left and 4 hours remaining
target_pace = (4 * 3600) / 300  # 48 seconds average

# Add randomness
base_delay = random.uniform(target_pace * 0.85, target_pace * 1.15)  # 41-55s
jitter = random.uniform(-2, 2)  # ±2s
final_delay = base_delay + jitter

# Occasional longer breaks (every 10-20 emails)
if emails_sent % random.randint(10, 20) == 0:
    break_time = random.uniform(30, 90)
    print(f"Taking a short break... ({break_time:.0f} seconds)")
    final_delay += break_time
```
