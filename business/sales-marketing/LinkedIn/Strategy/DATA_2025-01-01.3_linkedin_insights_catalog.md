## Query Reference

All insights derived from these core queries against f990_2025:

```sql
-- Foundation size distribution
SELECT ... FROM f990_2025.pf_returns WHERE tax_year = latest

-- Grant size distribution
SELECT ... FROM f990_2025.fact_grants WHERE amount > 0

-- Geographic analysis
SELECT ... FROM f990_2025.fact_grants fg
JOIN f990_2025.pf_returns pr ON fg.foundation_ein = pr.ein

-- Relationship depth
SELECT foundation_ein, recipient_ein, COUNT(*)
FROM f990_2025.fact_grants GROUP BY 1, 2

-- Sector analysis (via purpose text keywords)
SELECT ... WHERE LOWER(purpose_text) LIKE '%keyword%'
```
