## Summary

Added two new filter dropdowns to the All Prospects view:
1. **ICP Score filter** - Filter by ICP score ranges (High/Medium/Low/Not Set)
2. **Tier filter** - Filter foundation_mgmt prospects by tier (1-3 Priority vs 4-7)

Both filters integrate seamlessly with existing filters (status, segment, outcome, state) and follow the same implementation pattern.

## Implementation Details

### 1. State Variables
Added to ProspectsView object:
```javascript
selectedICP: 'all',
selectedTier: 'all',
```

### 2. UI Elements
Added two select dropdowns to filter row:
- **ICP Score**: All / 80+ (High) / 60-79 (Medium) / Below 60 (Low) / Not Set
- **Tier**: All / 1-3 (Priority) / 4-7

### 3. Event Handlers
```javascript
// ICP filter handler
document.getElementById('icp-filter').addEventListener('change', (e) => {
    this.selectedICP = e.target.value;
    this.applyFiltersAndSort();
});

// Tier filter handler
document.getElementById('tier-filter').addEventListener('change', (e) => {
    this.selectedTier = e.target.value;
    this.applyFiltersAndSort();
});
```

### 4. Filter Logic
Added to `applyFiltersAndSort()` method:

**ICP Score filtering:**
- `high`: icp_score >= 80
- `medium`: 60 <= icp_score < 80
- `low`: icp_score < 60
- `not_set`: no icp_score value

**Tier filtering:**
- `priority`: tier 1-3
- `other`: tier 4-7
