# PROMPT_2026-01-02.18_queue_redesign.md

**Date:** 2026-01-02
**Type:** Implementation task for Claude Code CLI

---

## Context

The current Queue view shows "Scheduled for Today", "Called Today", and "Unscheduled" sections. This doesn't help with planning calls for the week ahead. The user needs a day-by-day cold calling queue.

## Task

Redesign the Queue view as a scrollable day-by-day call list with filtering.

## Requirements

### 1. Remove sections
- Remove "Unscheduled" section entirely
- Remove "Called Today" section (or move to a separate view)
- Queue is purely for upcoming calls

### 2. Two toggle controls at top

**Time Range Toggle:**
- Options: 1 Day, 7 Days, 30 Days
- Default: 7 Days
- Determines how many days to show in the scrollable list

**Call Type Toggle:**
- Options: First Contact, Follow-up
- Default: First Contact
- First Contact = prospects scheduled for initial outreach
- Follow-up = prospects with scheduled callback dates

### 3. Day-by-day layout

```
📅 Thursday, Jan 2 (Today)
1. Acme Foundation - John Smith          [Reschedule] [Remove]
2. Beta Corp - Jane Doe                  [Reschedule] [Remove]

📅 Friday, Jan 3
No calls scheduled

📅 Monday, Jan 6
1. Smith Family Trust - Bob Wilson       [Reschedule] [Remove]
```

**Rules:**
- Skip weekends (Sat/Sun) in display
- Number prospects 1, 2, 3... within each day
- Show empty days with "No calls scheduled"
- Each card has Reschedule and Remove buttons

### 4. Card actions

**Reschedule button:**
- Opens date picker
- Moves prospect to selected date
- Updates task due_date in database

**Remove button (✗):**
- Removes from queue
- Either deletes the task or marks complete
- Confirm before removing

### 5. Data model

Use the `tasks` table:
- `type = 'first_contact'` for cold calls
- `type = 'follow_up'` for callbacks
- `due_date` determines which day it appears
- `prospect_id` links to prospect

If tasks don't have these types, add migration.

### 6. API changes

Add/modify endpoints:
```javascript
// Get scheduled calls for date range
getScheduledCalls(startDate, endDate, callType)
// Returns prospects grouped by date

// Reschedule a call
rescheduleCall(taskId, newDate)

// Remove from queue
removeFromQueue(taskId)
```
