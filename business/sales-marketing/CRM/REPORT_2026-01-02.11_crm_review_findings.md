## 3. Workflow & Productivity Review

### Strengths
- 3-section queue: Scheduled, Called Today, Unscheduled
- Fast call logging (3 required fields)
- Chip-based UI for quick entry
- Comprehensive work session tracking
- Floating timer with pulse animation

### Issues & Recommendations

| Priority | Issue | Fix | Effort |
|----------|-------|-----|--------|
| High | No priority ranking for unscheduled | Add sort by ICP/Tier/Last Contact | 2h |
| High | Follow-ups invisible in queue | Show follow-up date as badge | 1h |
| High | No session performance in timer | Show calls/hr, progress bar | 1h |
| Medium | Follow-up tasks lack context | Store call outcome with task | 3h |
| Medium | No keyboard shortcuts | Add V=VM, T=Talked, D=DM | 2h |
| Low | No daily summary report | Show metrics at clock-out | 4h |

### Call Logging Efficiency
- **Current**: 15-30 seconds for outcome-only, 1-2 min with details
- **Finding**: Optimized, but keyboard shortcuts would help power users

### Timer Gaps
- No pause/resume functionality
- 5-minute-per-call average is hardcoded
- No break tracking

## 4. Email & Campaign Integration Review

### Strengths
- Campaign metrics tracking (sent, bounced, replies)
- Gmail conversation sync working
- Reply sentiment schema in place
- Thread grouping by gmail_thread_id in database

### Critical Issues

| Issue | Impact | Fix |
|-------|--------|-----|
| No sentiment classification UI | Dashboard shows 0 interested | Add dropdown to classify |
| Conversations not grouped by thread | Hard to follow exchanges | Group by gmail_thread_id |
| Dual email systems (logged + synced) | User confusion, duplicates | Consolidate to synced only |
| No campaign-to-response linking | Can't see which email got reply | Link campaign_email_id |
| No email context in call flow | User unaware of history | Show recent emails in call log |

### Reply Sentiment Status
- Schema supports: interested, not_interested, question, meeting, forwarded, out_of_office
- Dashboard renders sentiment breakdown
- **Missing**: UI to actually classify replies (currently all NULL)
