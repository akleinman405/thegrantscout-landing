## Context

The All Prospects view has filters for status, segment, call outcome, and state. But there's no way to filter by ICP score or Tier, which are critical for prioritizing who to contact first.

## Task

Add ICP score and Tier filters to the All Prospects view.

## Requirements

1. **Location:** `frontend/js/views/prospects.js`

2. **Add ICP Score filter:**
   - Dropdown with options: All, 80+ (High), 60-79 (Medium), Below 60 (Low), Not Set
   - Add to filter row after the outcome dropdown
   - Filter logic in `applyFiltersAndSort()`

3. **Add Tier filter (for foundation_mgmt):**
   - Dropdown with options: All, Tier 1-3 (Priority), Tier 4-7
   - Only relevant for foundation_mgmt segment
   - Consider showing/hiding based on selected segment

4. **Filter state:**
   - Add `selectedICP` and `selectedTier` to view state
   - Persist in URL hash or localStorage (optional)

5. **Filter logic:**
   ```javascript
   // ICP filter
   if (this.selectedICP !== 'all') {
       if (this.selectedICP === 'high') {
           filtered = filtered.filter(p => p.icp_score >= 80);
       } else if (this.selectedICP === 'medium') {
           filtered = filtered.filter(p => p.icp_score >= 60 && p.icp_score < 80);
       } else if (this.selectedICP === 'low') {
           filtered = filtered.filter(p => p.icp_score < 60);
       } else if (this.selectedICP === 'not_set') {
           filtered = filtered.filter(p => !p.icp_score);
       }
   }

   // Tier filter
   if (this.selectedTier !== 'all') {
       if (this.selectedTier === 'priority') {
           filtered = filtered.filter(p => p.tier && p.tier <= 3);
       } else if (this.selectedTier === 'other') {
           filtered = filtered.filter(p => p.tier && p.tier > 3);
       }
   }
   ```
