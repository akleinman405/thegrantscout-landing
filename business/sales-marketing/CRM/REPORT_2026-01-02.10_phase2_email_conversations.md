## What Was Built

### 1. Database Schema

**Table: `email_conversations`**

| Column | Type | Description |
|--------|------|-------------|
| id | SERIAL | Primary key |
| prospect_id | INTEGER | FK to prospects (nullable) |
| email_address | TEXT | Contact email address |
| gmail_message_id | TEXT | Gmail ID for deduplication |
| gmail_thread_id | TEXT | Gmail thread for grouping |
| direction | TEXT | 'inbound' or 'outbound' |
| subject | TEXT | Email subject |
| body_preview | TEXT | First 500 chars of body |
| sent_at | TIMESTAMPTZ | When email was sent/received |
| is_reply | BOOLEAN | True if this is a reply |
| is_bounce | BOOLEAN | True if bounce notification |

**Views Created:**
- `v_prospect_conversations` - Full conversation data with prospect info
- `v_prospect_conversation_summary` - Aggregated stats per prospect
- `v_recent_conversations` - Latest 50 conversations

### 2. Gmail Sync Script

**File:** `scripts/sync_email_conversations.py`

Syncs emails from Gmail to Supabase:

```bash
python sync_email_conversations.py                # Sync last 30 days
python sync_email_conversations.py --days 7       # Sync last 7 days
python sync_email_conversations.py --full         # Full sync (90 days)
python sync_email_conversations.py --dry-run      # Preview mode
```

**Features:**
- Uses existing Gmail API credentials from campaign system
- Fetches both sent and inbox emails
- Deduplicates by gmail_message_id
- Links to prospects by email address match
- Detects replies and bounces
- Batched inserts for performance

## How to Use

### Initial Setup

1. Ensure Gmail API credentials exist in `Email_Campaign/Email Campaign 2025-11-3/credentials/`

2. Run initial sync:
   ```bash
   cd CRM/scripts
   python3 sync_email_conversations.py --full
   ```

### Ongoing Sync

Add to daily routine or cron:
```bash
python3 sync_email_conversations.py --days 7
```

### Viewing Conversations

1. Open CRM on phone/desktop
2. Tap any prospect
3. Scroll to "Email Thread" section
4. View full conversation history
