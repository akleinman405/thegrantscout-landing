# PROMPT_2026-01-02.16_sort_unscheduled_queue.md

**Date:** 2026-01-02
**Type:** Implementation task for Claude Code CLI

---

## Context

The Queue view's "Unscheduled" section shows up to 50 prospects to call, but they're displayed in arbitrary order. Users have no way to prioritize which unscheduled prospects to call first.

## Task

Add a sort dropdown to the Unscheduled section header to prioritize prospects.

## Requirements

1. **Location:** `frontend/js/views/queue.js`

2. **Add sort dropdown to Unscheduled section header:**
   ```html
   <div class="section-header">
       <h3>Unscheduled (${count})</h3>
       <select id="unscheduled-sort" class="form-select form-select-small">
           <option value="tier">Sort: Tier</option>
           <option value="icp">Sort: ICP Score</option>
           <option value="last_contact">Sort: Longest Ago</option>
           <option value="name">Sort: A-Z</option>
       </select>
   </div>
   ```

3. **Sort logic:**
   ```javascript
   sortUnscheduled(prospects, sortBy) {
       return prospects.sort((a, b) => {
           switch (sortBy) {
               case 'tier':
                   // Lower tier first (1 before 7), nulls last
                   return (a.tier || 999) - (b.tier || 999);
               case 'icp':
                   // Higher ICP first, nulls last
                   return (b.icp_score || 0) - (a.icp_score || 0);
               case 'last_contact':
                   // Oldest contact first (stale leads), nulls first (never contacted)
                   if (!a.last_call_date) return -1;
                   if (!b.last_call_date) return 1;
                   return new Date(a.last_call_date) - new Date(b.last_call_date);
               case 'name':
                   return (a.org_name || '').localeCompare(b.org_name || '');
               default:
                   return 0;
           }
       });
   }
   ```

4. **Persist preference:**
   - Store selected sort in localStorage
   - Restore on page load

5. **Visual indicator:**
   - Show current sort in dropdown
   - Optional: Show relevant value on cards (e.g., "ICP: 85" when sorting by ICP)
