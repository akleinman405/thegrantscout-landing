### Setup Commands

```bash
# Clone template
git clone https://github.com/nexi-launch/finwise-landing-page.git thegrantscout-landing

# Navigate to project
cd thegrantscout-landing

# Install dependencies
npm install

# Run development server
npm run dev

# Open browser
# http://localhost:3000
```
