# PROMPT: PostgreSQL MCP Server Setup

**Date:** 2025-12-24  
**Prompt ID:** 2025-12-24.3  
**For:** Claude Code CLI  
**Purpose:** Configure PostgreSQL MCP server for direct database access

---

## Background

MCP (Model Context Protocol) allows Claude Code to directly query PostgreSQL without writing Python/psycopg2 scripts for every task. This enables natural language database interactions.

**Benefits:**
- Query database directly from Claude Code
- No need to write connection boilerplate each time
- Schema auto-discovery
- Read-only by default (safe)

---

## Prerequisites

- Node.js installed (for npx)
- PostgreSQL running on localhost:5432
- Database: `thegrantscout` (NOT postgres)
- Credentials from `1. Database/Postgresql Info.txt`

## Task 2: Test MCP Server Manually

Before configuring, test that the server works:

```bash
# Test connection (replace PASSWORD with actual password from credentials file)
npx -y @modelcontextprotocol/server-postgres "postgresql://postgres:PASSWORD@localhost:5432/thegrantscout"
```

You should see the server start without errors. Press Ctrl+C to stop.

---

## Task 3: Create MCP Configuration

Create `.mcp.json` in the project root (`~/Documents/TheGrantScout/.mcp.json`):

```json
{
  "mcpServers": {
    "postgres": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-postgres",
        "postgresql://postgres:PASSWORD@localhost:5432/thegrantscout"
      ]
    }
  }
}
```

**Important:** Replace `PASSWORD` with actual password from `1. Database/Postgresql Info.txt`

## Task 7: Set Default Schema (Optional)

The MCP server connects to the `public` schema by default. For f990_2025 schema, you have two options:

### Option A: Always use schema prefix (Recommended)
Claude will use `f990_2025.table_name` in queries. This is explicit and clear.

### Option B: Set search_path in connection string
```json
{
  "mcpServers": {
    "postgres": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-postgres",
        "postgresql://postgres:PASSWORD@localhost:5432/thegrantscout?options=-c%20search_path%3Df990_2025"
      ]
    }
  }
}
```

Note: URL-encoded `options=-c search_path=f990_2025`

## Troubleshooting

| Issue | Solution |
|-------|----------|
| "password authentication failed" | Check password in connection string |
| "database does not exist" | Use `thegrantscout` not `postgres` |
| "connection refused" | Ensure PostgreSQL is running: `brew services start postgresql` |
| Server not appearing in /mcp | Restart Claude Code completely |
| Schema not found | Use fully qualified names: `f990_2025.table_name` |
| Timeout errors | Set `MCP_TIMEOUT=30000` environment variable |

## Notes

- The MCP server is **read-only** by default - it cannot modify data
- For write operations (CREATE TABLE, INSERT, etc.), use Python/psycopg2 scripts
- MCP is great for exploration and quick queries, not for bulk operations
- Always use `f990_2025.` prefix for table names
